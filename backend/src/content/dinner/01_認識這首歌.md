# 01 — 認識這首歌

> 在拿起吉他之前，先用「耳朵」和「腦袋」學會這首歌。

---

## 為什麼要先「認識」一首歌？

很多初學者拿到和弦譜就埋頭苦練，結果彈了半天覺得「怎麼聽起來不太對」。原因是：**你還沒有在腦中建立這首歌的「地圖」。**

想像一下：你要開車去一個沒去過的地方，如果先看過地圖、知道大概的路線和轉彎點，上路後就不容易迷路。學一首歌也是一樣的道理。

在這一篇中，我們會建立幾件事情：

1. 這首歌的**基本資訊**
2. 它的**調性**是什麼、為什麼重要
3. 歌曲的**結構長什麼樣子**
4. **情緒和力度**是怎麼變化的

## 2. 調性分析 — 為什麼要理解調性？

### 什麼是「調性」？

簡單來說，調性決定了一首歌使用哪些音符和和弦。就像畫畫時選用的「色盤」，調性就是這首歌的「音色盤」。

### 這首歌的調性

| 概念       | 說明                                                                |
| ---------- | ------------------------------------------------------------------- |
| **原調**   | A♭ 大調（A-flat Major）— 這是原曲的實際音高                         |
| **演奏調** | C 大調 — 我們用 Capo 夾第 1 格，就可以用 C 調的和弦彈出 A♭ 調的音高 |

### 為什麼要用 Capo？

A♭ 大調的和弦（A♭、D♭、E♭、Fm）在吉他上都是**封閉和弦（Barre Chord）**，需要食指橫按整個琴格，對初學者來說門檻很高。

使用 **Capo（變調夾）** 的概念：

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  原調 A♭ 的和弦       Capo 1 + C 調指型              │
│  ─────────────        ──────────────────             │
│  A♭  （封閉和弦）  →   G   （開放和弦）✓ 好按！      │
│  D♭  （封閉和弦）  →   C   （開放和弦）✓ 好按！      │
│  E♭  （封閉和弦）  →   D   （開放和弦）✓ 好按！      │
│  Fm  （封閉和弦）  →   Em  （開放和弦）✓ 好按！      │
│                                                      │
│  → Capo 等於幫你「升高琴頸的起點」                    │
│    你按的指型不變，但實際音高升了半音                  │
│                                                      │
└──────────────────────────────────────────────────────┘
```

> [!NOTE]
> **Capo 的本質：** 變調夾夾在第 N 格，等於所有音升高 N 個半音。
> 夾第 1 格 → 升高 1 個半音。C 大調升 1 個半音 = D♭ 大調。
> 但因為 G/B → Cadd9 → D → Em 的進行等效於 A♭ 大調中的
> A♭/C → D♭add9 → E♭ → Fm，所以結果就是原調 A♭。

## 3. 節拍與速度

| 項目 | 數值                         | 感覺                   |
| ---- | ---------------------------- | ---------------------- |
| 拍號 | 4/4 拍（最常見的流行歌拍號） | 每小節 4 拍，每拍等長  |
| BPM  | 103                          | 中板偏慢，像散步的速度 |

### 什麼是 BPM？

BPM = Beats Per Minute（每分鐘幾拍）。你可以把它想像成「心跳速度」：

- 60 BPM = 每秒 1 拍（很慢，像安靜休息時的心跳）
- 103 BPM = 每秒約 1.7 拍（像在公園散步時的心跳）
- 120 BPM = 每秒 2 拍（精神抖擻地走路）
- 140+ BPM = 快速（像跑步時的心跳）

**練習時的建議：** 先用 60～70 BPM 練，等流暢後再逐步提升到 103 BPM。

## 4. 歌曲結構圖

流行歌曲通常有固定的「段落結構」。了解結構就像拿到了一張地圖，讓你知道「現在在哪、接下來要去哪」。

### 段落名稱對照

| 英文名         | 中文名           | 功能                                   |
| -------------- | ---------------- | -------------------------------------- |
| **Intro**      | 前奏             | 營造氣氛，引導聽眾進入歌曲             |
| **Verse**      | 主歌（A 段）     | 敘事段落，旋律相對平穩                 |
| **Pre-Chorus** | 預備副歌（B 段） | 承上啟下，為副歌的爆發做鋪墊           |
| **Chorus**     | 副歌             | 全曲最記憶點最高的段落，旋律最強烈     |
| **Bridge**     | 橋段             | 出現在後段，帶來新鮮感，通常是情感高峰 |
| **Outro**      | 尾奏             | 收尾，讓歌曲自然結束                   |

### 晩餐歌的完整結構

```

 │Intro│→│ Verse 1 │→│Pre-Cho 1│→│Chorus 1 │→│ Verse 2 │→
 │前奏 │ │ 主歌一  │ │預備副歌 │ │ 副歌一  │ │ 主歌二  │
 │琶音 │ │  琶音   │ │琶→刷弦  │ │  刷弦   │ │  琶音   │
```

## 5. 力度與情緒走向

這首歌的情緒不是從頭到尾一樣強烈的，而是有起有伏，像呼吸一樣。理解這一點非常重要，因為彈吉他不只是「按對和弦、彈對節奏」，更重要的是**傳達音樂的情感**。

### 力度術語速查

這些是音樂中表示力度強弱的標準術語（來自義大利語）：

| 符號 | 義大利語    | 意思   | 類比             |
| ---- | ----------- | ------ | ---------------- |
| pp   | pianissimo  | 非常弱 | 像在耳邊輕聲細語 |
| p    | piano       | 弱     | 像在對面輕聲說話 |
| mp   | mezzo piano | 中弱   | 正常的說話音量   |
| mf   | mezzo forte | 中強   | 稍微提高聲音     |
| f    | forte       | 強     | 像在對一群人說話 |
| ff   | fortissimo  | 非常強 | 像在大聲吶喊     |

### 晩餐歌的力度曲線

### 解讀力度曲線

1. **前奏 & 主歌**（pp → p）：像在夜晚安靜的房間裡，對著一個人輕聲訴說。吉他用琶音輕輕撥弦。

2. **預備副歌**（mp → f）：語氣開始激動，情緒醞釀。吉他從琶音逐漸過渡到刷弦。

3. **副歌**（f）：累積的情感爆發出來。吉他改為有力的刷弦。

4. **橋段**（ff）：全曲的最高潮。歌詞內容是「結果只有你一個人留在我身邊」這樣觸及核心的告白。吉他刷弦最猛烈。

5. **尾奏**（pp）：情緒漸漸平復，回到開頭的琶音，像潮水退去。

> [!IMPORTANT]
> **外行彈對錯，內行彈強弱。**
> 能把力度變化做好的人，即使技巧不夠華麗，聽起來也會比「技巧很好但從頭到尾一樣大聲」的人動聽得多。記住這個觀念，從現在開始就刻意練習力度控制。

## 6. 第一步行動：聽歌

在繼續往下一篇之前，請你做一件事：

### 🎧 至少完整聽三遍原曲

1. **第一遍：隨意聽** — 不分析，純粹感受這首歌帶給你的情緒。
2. **第二遍：注意結構** — 試著聽出段落的分界。什麼時候音樂「變強了」？什麼時候「安靜下來了」？
3. **第三遍：注意吉他** — 把注意力放在伴奏上。聽聽什麼時候是「一顆一顆的音」（琶音），什麼時候是「刷的一聲」（刷弦）。

當你能在腦中「哼出」整首歌的旋律和段落切換時，你就準備好進入下一步了。
