## 1. 你需要準備的東西

### 🎸 一把民謠吉他（Acoustic Guitar）

- 使用**鋼弦民謠吉他**，不是古典吉他（尼龍弦）。
- 民謠吉他的聲音比較明亮、清脆，更貼近這首歌的原曲音色。
- 如果你手邊只有古典吉他或電吉他，也可以先用來練習和弦與結構，不影響學習。

### 🔧 變調夾（Capo）

- 這首歌我們需要把 Capo 夾在**第 1 格**。
- 建議購買彈簧式（Trigger Capo）或螺旋式（Screw Capo），兩種都好用。

### 🎵 調音器（Tuner）

可以是以下任一種：

- **夾式調音器**（夾在琴頭上，最方便）
- **手機 App**（推薦：GuitarTuna ── 免費，介面直覺）
- **線上調音器**（Google 搜尋 "guitar tuner"）

### 📱 節拍器（Metronome）

- 手機 App 推薦：Pro Metronome、Soundbrenner
- 或直接 Google 搜尋 "metronome"，瀏覽器就能用
- 這首歌的目標速度：**103 BPM**

### 🎧 原曲音源

- 串流平台搜尋「tuki. 晩餐歌」
- YouTube 搜尋「晩餐歌 tuki.」

## 2. 吉他調音

### 標準調音（Standard Tuning）

吉他有 6 根弦，從最粗到最細，標準音高如下：

```
粗 ←────────────────────────────→ 細

 第 6 弦    5     4     3     2    第 1 弦
  ┃        ┃     ┃     ┃     ┃     ┃
  E2       A2    D3    G3    B3    E4
  ┃        ┃     ┃     ┃     ┃     ┃
  低音 E    A     D     G     B    高音 E
```

### 記憶口訣

> **E**at **A**ll **D**ay, **G**et **B**ig **E**asy
> （吃一整天，輕鬆變大）

或者用中文記：**「衣 阿 地 雞 逼 衣」**（模擬英文發音）

### 調音步驟

1. 打開調音器。
2. 撥動一根弦，調音器會顯示目前的音名和偏差。
3. **指針偏左**（偏低）→ 旋轉弦鈕讓弦**繃緊**（音升高）。
4. **指針偏右**（偏高）→ 旋轉弦鈕讓弦**放鬆**（音降低）。
5. 指針在正中央、燈號變綠 → 這根弦調好了。
6. 依序把 6 根弦全部調完。

> [!CAUTION]
> **調弦時要慢慢轉！** 如果轉太快、一次繃太緊，弦可能會斷裂。尤其是第 1、2 弦（最細的兩根），要特別小心。

## 3. Capo（變調夾）的使用

### Capo 是什麼？

Capo 是一個夾在琴格上的工具，它的作用等同於**在那個位置多了一根手指幫你全部橫按住**。夾上 Capo 後，吉他的「起點」就從開放弦變成了 Capo 所在的位置。

### Capo 的原理

```
沒有 Capo 的情況：

  琴枕（Nut）
  ┃
  ┃  第1格  第2格  第3格  第4格
  ┃    │      │      │      │
  ╠════╪══════╪══════╪══════╪════ ← 弦
  ┃    │      │      │      │
  ┃  空弦 = E（以第 1 弦為例）

─────────────────────────────────────

Capo 夾在第 1 格的情況：

       Capo
  ┃    ┃
  ┃    ┃  第2格  第3格  第4格
  ┃    ┃    │      │      │
  ╠════╬════╪══════╪══════╪════ ← 弦
  ┃    ┃    │      │      │
       ┃
       ┗━► Capo 後的世界：
           這裡變成新的「第 1 格」
           空弦音 = F（比原本的 E 高半音）
```

### 正確安裝 Capo 的方式

安裝位置會直接影響音準和音色，以下是關鍵要點：

```
              ← 琴頭方向          琴身方向 →

                    ╔═══╗
        ┃    金屬品格 ║   ║ 金屬品格    ┃
        ┃           ║   ║             ┃
        ┃     ❌    ║ ✅ ║    ❌       ┃
        ┃    太靠左  ║正確║  太靠右     ┃
        ┃           ║位置║             ┃
        ┃           ╚═══╝             ┃

  ✅ 正確：Capo 緊貼金屬品格的右側（靠琴身方向），但不壓在品格上
  ❌ 錯誤：壓在品格正上方（會打品、雜音）
  ❌ 錯誤：離品格太遠（音會不準、需要太大力）
```

### 安裝後的檢查

夾好 Capo 之後，**逐一撥動每根弦**，確認：

| 檢查項目  | 正常               | 異常（需調整）           |
| --------- | ------------------ | ------------------------ |
| 音色      | 每根弦聲音清脆飽滿 | 有悶悶的、嗡嗡的雜音     |
| 音準      | 調音器顯示正確音高 | 某些弦偏高或偏低         |
| Capo 位置 | 平行於品格         | 歪斜（導致各弦壓力不均） |

> [!TIP]
> **Capo 夾上後建議重新微調音準。** 便宜的 Capo 可能會把弦壓得太用力，導致音稍微偏高。如果你的 Capo 是螺旋式的，可以調整夾緊程度來改善這個問題。

## 4. Capo 夾法對應的音高

了解 Capo 在不同位置對應的實際音高，有助於未來舉一反三：

| Capo 位置 | 你按 C 和弦 = 實際是 | 你按 G 和弦 = 實際是 |
| --------- | -------------------- | -------------------- |
| 無 Capo   | C                    | G                    |
| 第 1 格   | D♭ (C#)              | A♭ (G#)              |
| 第 2 格   | D                    | A                    |
| 第 3 格   | E♭                   | B♭                   |
| 第 4 格   | E                    | B                    |
| 第 5 格   | F                    | C                    |

> 這首歌 Capo 在第 1 格，你按的 C 調和弦 → 實際聽到的是 A♭ 調。

## 5. 練習前的暖身

在正式練習之前，花 3～5 分鐘做手指暖身，可以有效降低受傷風險、提升練習效率。

### 暖身運動 1：手指張合

```
1. 把手掌攤開，手指盡量張開 → 維持 3 秒
2. 手指全部握拳 → 維持 3 秒
3. 重複 10 次
```

### 暖身運動 2：半音階爬格子

在吉他上依序按每一格，訓練左手手指的獨立性：

```
TAB 譜表示法：

e|--1--2--3--4--|
B|--1--2--3--4--|
G|--1--2--3--4--|
D|--1--2--3--4--|
A|--1--2--3--4--|
E|--1--2--3--4--|

說明：
- 從第 6 弦（最粗）開始
- 食指按第 1 格、中指按第 2 格、無名指按第 3 格、小指按第 4 格
- 每按一格就撥一次弦
- 一弦彈完換下一弦
- 全部彈完後可以從第 1 弦反過來彈回去

重點：
- 速度不重要，清晰度才重要
- 每個音都要清脆、沒有雜音
- 手指盡量「直立」按弦，不要碰到旁邊的弦
```

### 暖身運動 3：手腕旋轉

```
1. 雙手向前平伸
2. 手腕順時針旋轉 10 圈
3. 再逆時針旋轉 10 圈
4. 不要太用力，感覺手腕暖起來就好
```
