# 03 — 和弦全攻略

> 7 個和弦，從「知道怎麼按」到「知道為什麼這樣按」。

---

## 本篇目標

- ✅ 學會 7 個和弦的正確按法
- ✅ 理解每個和弦的「角色」（它在歌曲中扮演什麼功能）
- ✅ 掌握和弦之間的轉換路徑
- ✅ 建立和弦轉換的肌肉記憶

---

## 0. 讀譜基礎知識

在看和弦圖之前，先搞懂這些符號：

### 和弦圖的閱讀方式

```
        ← 琴頭方向

  粗弦                    細弦
  6弦  5弦  4弦  3弦  2弦  1弦
   E    A    D    G    B    e
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  ← 第 1 格（Capo 之後）
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  ← 第 2 格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  ← 第 3 格
   ┃    ┃    ┃    ┃    ┃    ┃
```

### 符號說明

| 符號 | 意思                           |
| ---- | ------------------------------ |
| `x`  | 不彈這根弦（右手不碰它）       |
| `o`  | 空弦彈奏（左手不按，直接撥弦） |
| `●`  | 左手手指按在這個位置           |

### 左手手指編號

```
        食指 ── 1
        中指 ── 2
      無名指 ── 3
        小指 ── 4

  （拇指不編號，拇指放在琴頸後方支撐）
```

> [!NOTE]
> **所有格數都是以 Capo 之後為基準。** 因為 Capo 在第 1 格，所以本文說的「第 1 格」實際上是吉他的第 2 格，但你不需要刻意去算，只要從 Capo 往下數就對了。

## 1. 主要和弦（全曲的骨架）

這 4 個和弦構成了這首歌最核心的進行（Verse 和 Chorus 的主體）。

---

### 1.1　G/B — 帶有低音 B 的 G 和弦

```
   6弦  5弦  4弦  3弦  2弦  1弦
    x    ●    o    o    o    ●
         2                   3
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪════╪════╪════╣  2格   5弦：中指(2)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪══●═╣  3格   1弦：無名指(3)
```

**按法逐弦說明：**

| 弦   | 按法            | 原因                                              |
| ---- | --------------- | ------------------------------------------------- |
| 6 弦 | x（不彈）       | 最低音是 B，不是 E，所以跳過                      |
| 5 弦 | 中指按第 2 格   | 這個位置是 **B 音** ── 這就是和弦名中「/B」的來源 |
| 4 弦 | 空弦            | D 音 ── G 和弦的組成音之一（5 度音）              |
| 3 弦 | 空弦            | G 音 ── 和弦的根音                                |
| 2 弦 | 空弦            | B 音 ── 和弦的 3 度音                             |
| 1 弦 | 無名指按第 3 格 | G 音 ── 跟 3 弦一樣，再多一個根音讓聲音更飽滿     |

**為什麼用 G/B 而不是普通 G？**

G/B 的「/B」叫做**轉位（Inversion）**，意思是「G 和弦，但最低音改成 B」。

在這首歌中，和弦進行是 `G/B → Cadd9 → D → Em`，它們的最低音分別是：

```
和弦：  G/B  →  Cadd9  →  D   →  Em
最低音：  B   →   C    →  D   →  E

低音走向：B → C → D → E  （半音＋全音＋全音，順階上行）
```

這條**上行的低音線**讓和弦進行聽起來非常流暢、有方向感。如果用普通 G，最低音就變成 G，這條線就斷掉了。

> [!TIP]
> **初學者替代方案：** 如果你一開始按 G/B 覺得彆扭，可以先用普通 G 和弦練習，等其他部分都熟了再回來替換。不影響學習進度。

---

### 1.2　Cadd9 — 掛九度的 C 和弦

```
   6弦  5弦  4弦  3弦  2弦  1弦
    x    ●    ●    o    ●    o
         3    2         4
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪══●═╪════╪════╣  2格   4弦：中指(2)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪════╪════╪══●═╣  3格   5弦：無名指(3)、2弦：小指(4)
```

**按法逐弦說明：**

| 弦   | 按法            | 音名               |
| ---- | --------------- | ------------------ |
| 6 弦 | x（不彈）       | —                  |
| 5 弦 | 無名指按第 3 格 | C                  |
| 4 弦 | 中指按第 2 格   | E                  |
| 3 弦 | 空弦            | G                  |
| 2 弦 | 小指按第 3 格   | D ← 這就是「add9」 |
| 1 弦 | 空弦            | E                  |

**什麼是「add9」？**

普通的 C 和弦由三個音組成：C（根音）、E（3 度）、G（5 度）。

**Cadd9** 多加了一個 **D 音**，D 是 C 的第 9 度音（其實就是第 2 度音飛高一個八度）。

聽覺效果：比普通 C 更空靈、帶有一種「未完成」的懸浮感，非常適合這首歌溫柔又帶感傷的氛圍。

> **替代方案：** 想簡化的話，把小指（2 弦第 3 格）換成食指按 2 弦第 1 格 → 就變成普通 **C 和弦**，完全可用。

---

### 1.3　D — D 大調和弦

```
   6弦  5弦  4弦  3弦  2弦  1弦
    x    x    o    ●    ●    ●
                   1    3    2
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪══●═╪════╪══●═╣  2格   3弦：食指(1)、1弦：中指(2)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪══●═╪════╣  3格   2弦：無名指(3)
```

**按法逐弦說明：**

| 弦   | 按法            | 音名       |
| ---- | --------------- | ---------- |
| 6 弦 | x（不彈）       | —          |
| 5 弦 | x（不彈）       | —          |
| 4 弦 | 空弦            | D（根音）  |
| 3 弦 | 食指按第 2 格   | A（5 度）  |
| 2 弦 | 無名指按第 3 格 | D（根音）  |
| 1 弦 | 中指按第 2 格   | F#（3 度） |

> [!WARNING]
> **D 和弦最常見的錯誤：** 不小心碰到第 5 或第 6 弦。這兩根弦的音不屬於 D 和弦，彈到會聽起來很「混濁」。右手撥弦時要注意控制範圍。

---

### 1.4　Em — E 小調和弦

```
   6弦  5弦  4弦  3弦  2弦  1弦
    o    ●    ●    o    o    o
         2    3
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪══●═╪════╪════╣  2格   5弦：中指(2)、4弦：無名指(3)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  3格
```

**按法逐弦說明：**

| 弦   | 按法            | 音名                                    |
| ---- | --------------- | --------------------------------------- |
| 6 弦 | 空弦            | E（根音）                               |
| 5 弦 | 中指按第 2 格   | B（5 度）                               |
| 4 弦 | 無名指按第 2 格 | E（根音）                               |
| 3 弦 | 空弦            | G（♭3 度 ← 讓它聽起來是「小調」的關鍵） |
| 2 弦 | 空弦            | B（5 度）                               |
| 1 弦 | 空弦            | E（根音）                               |

Em 是吉他上最容易按的和弦之一 —— 只需要兩根手指。它在歌曲中扮演的是**悲傷、感傷**的角色，恰好呼應歌詞的情感。

## 2. 輔助和弦（Pre-Chorus 和 Bridge 使用）

---

### 2.1　Am — A 小調和弦

```
   6弦  5弦  4弦  3弦  2弦  1弦
    x    o    ●    ●    ●    o
              3    2    1
   ╠════╪════╪════╪════╪══●═╣  1格   2弦：食指(1)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪══●═╪════╣  2格   3弦：中指(2)
   ╠════╪════╪══●═╪════╪════╣       4弦：無名指(3) （也在第2格）
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  3格
```

**按法：** 食指(1)→2 弦第 1 格、中指(2)→3 弦第 2 格、無名指(3)→4 弦第 2 格。

---

### 2.2　C — C 大調和弦

```
   6弦  5弦  4弦  3弦  2弦  1弦
    x    ●    ●    o    ●    o
         3    2         1
   ╠════╪════╪════╪════╪══●═╣  1格   2弦：食指(1)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪══●═╪════╪════╣  2格   4弦：中指(2)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪════╪════╪════╣  3格   5弦：無名指(3)
```

**按法：** 食指(1)→2 弦第 1 格、中指(2)→4 弦第 2 格、無名指(3)→5 弦第 3 格。

> [!NOTE]
> **Am 和 C 的關係：** 仔細看，Am 和 C 的指型非常像 —— 食指都在 2 弦第 1 格。轉換時，食指不動，中指和無名指各往低音方向移一根弦就好了。這是一個非常高效的轉換。

---

### 2.3　G — G 大調和弦

```
   6弦  5弦  4弦  3弦  2弦  1弦
    ●    ●    o    o    o    ●
    2    1                   3
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪════╪════╪════╣  2格   5弦：食指(1)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠══●═╪════╪════╪════╪══●═╣  3格   6弦：中指(2)、1弦：無名指(3)
```

**按法：** 食指(1)→5 弦第 2 格、中指(2)→6 弦第 3 格、無名指(3)→1 弦第 3 格。

## 3. 和弦角色總覽

在音樂理論中，每個和弦在一個調性裡都有特定的「功能」。以下是這首歌（C 大調）中各和弦的角色：

| 和弦      | 級數        | 功能         | 情緒感受                     |
| --------- | ----------- | ------------ | ---------------------------- |
| C / Cadd9 | I（一級）   | **主和弦**   | 安穩、回家的感覺             |
| D         | II（二級）  | **屬預備**   | 期待感、「接下來要到高潮了」 |
| Em        | iii（三級） | **代理主**   | 柔和的哀傷                   |
| G / G/B   | V（五級）   | **屬和弦**   | 張力、推進力「要回家了」     |
| Am        | vi（六級）  | **關係小調** | 正式的哀傷、情感深沉         |

> 你不需要記住這些理論，但理解它們有助於你感受和弦帶來的「情緒移動」。

## 4. 和弦轉換訓練

和弦轉換是初學者最大的瓶頸，以下是按照這首歌的需求設計的專項練習。

### 4.1 轉換原則

1. **找共同手指** — 如果兩個和弦之間有同一根手指在附近的位置，那根手指當「錨點」，其他手指再移動。
2. **同時到達** — 不要一根一根手指放下去，練習所有手指同時落到指板上。
3. **先慢後快** — 節拍器從 40 BPM 開始，每次到一個和弦就刷一下，在拍子上精準到達。

### 4.2 本曲關鍵轉換路徑

#### 路徑 A：G/B → Cadd9 → D → Em（Verse 進行）

```
G/B                 Cadd9               D                   Em
5弦2格(中指)        5弦3格(無名指)      無                   5弦2格(中指)
無                   4弦2格(中指)        3弦2格(食指)         4弦2格(無名指)
無                   2弦3格(小指)        2弦3格(無名指)       無
1弦3格(無名指)      無                   1弦2格(中指)         無

轉換技巧：
G/B → Cadd9 ：中指從 5弦2格 滑到 5弦3格  → 其他手指跟上
Cadd9 → D   ：所有手指重新排列（無共同點，需要整組放開重按）
D → Em      ：食指放開，中指無名指平移到 5、4 弦第 2 格
```

> [!TIP]
> **最難的轉換是 Cadd9 → D**，因為完全沒有共同手指。
> 解決方法：反覆練習這組轉換 50 次以上，讓身體記住移動路徑。

#### 路徑 B：Am → C（Pre-Chorus 常用）

```
Am          →          C
2弦1格(食指)    ✦ 不動 ✦    2弦1格(食指)
3弦2格(中指)    往低音移     4弦2格(中指)
4弦2格(無名指)  往低音移     5弦3格(無名指)
```

食指完全不動，中指和無名指各往粗弦方向移一格。這是最容易的轉換之一。

#### 路徑 C：C → G（Pre-Chorus 中段）

```
C           →          G
2弦1格(食指)            5弦2格(食指)
4弦2格(中指)            6弦3格(中指)
5弦3格(無名指)          1弦3格(無名指)
```

手型大幅移動，需要多練。技巧：無名指從 5 弦第 3 格直接跳到 1 弦第 3 格（格數不變、只換弦）。

## 5. 練習方法

### 練習 1：單和弦清晰度測試

按住一個和弦後，從最粗的弦（或該和弦最低要彈的弦）開始，**一根一根撥弦**。每根弦都應該發出清脆的聲音：

- ✅ 清脆 → 這根弦 OK
- ❌ 悶悶的 → 手指可能壓力不夠，或碰到了旁邊的弦
- ❌ 嗡嗡聲 → 手指可能太靠近品格（金屬桿），往稍遠的方向移一點

**目標：** 每個和弦的每根弦都能清脆發聲。

### 練習 2：兩和弦來回切換（1 分鐘計時）

1. 設定計時器 1 分鐘。
2. 選兩個和弦，在拍子上來回切換。
3. 每成功切換一次，計 1 分。
4. 目標：每分鐘 30 次以上。

建議練習順序（由易到難）：

| 順序 | 組合        | 難度 |
| ---- | ----------- | ---- |
| 1    | Em ↔ Am     | ★☆☆  |
| 2    | Am ↔ C      | ★☆☆  |
| 3    | G/B ↔ Cadd9 | ★★☆  |
| 4    | D ↔ Em      | ★★☆  |
| 5    | Cadd9 ↔ D   | ★★★  |
| 6    | C ↔ G       | ★★★  |

### 練習 3：四和弦循環（配合節拍器）

當上面的兩和弦切換都達到每分鐘 30 次後，開始練習四和弦循環：

```
節拍器設定：40 BPM → 逐步加到 103 BPM

循環 1（Verse 進行）：
| G/B（4拍）| Cadd9（4拍）| D（4拍）| Em（4拍）| → 重複

循環 2（Pre-Chorus 進行）：
| Am（4拍）| C（4拍）| G（4拍）| D（4拍）| → 重複
```

每個和弦 4 拍 = 全音符長度。在每個拍子的第 1 拍刷一下，給自己足夠的時間切換。
