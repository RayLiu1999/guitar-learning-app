# 04 — 右手技巧篇：琶音與刷弦

> 左手決定「彈什麼音」，右手決定「怎麼彈出來」。

---

## 本篇目標

- ✅ 掌握琶音（Arpeggio）的基本與進階模式
- ✅ 掌握刷弦（Strumming）的節奏型態
- ✅ 學會琶音→刷弦的過渡技巧
- ✅ 理解「什麼段落用什麼彈法」的音樂邏輯

---

## 1. 右手基礎姿勢

### 手指代號（國際通用）

這些代號來自西班牙語，是全世界吉他手共通的「語言」：

| 代號  | 西班牙語 | 中文   | 負責的弦                   |
| ----- | -------- | ------ | -------------------------- |
| **p** | pulgar   | 拇指   | 6 弦、5 弦、4 弦（低音弦） |
| **i** | índice   | 食指   | 3 弦                       |
| **m** | medio    | 中指   | 2 弦                       |
| **a** | anular   | 無名指 | 1 弦                       |

> **小指**在指彈中通常不使用，可以輕放或自然懸空。

### 右手放置位置

```
                    琴橋 ←──── 音孔 ────→ 琴頸
                      │         │          │
                      │    ★ 正確位置 ★    │
                      │    (音孔正上方      │
                      │     偏琴橋一點)     │
                      │         │          │

   太靠琴橋 → 聲音乾硬、金屬感重
   在音孔上方 → 聲音飽滿、溫暖（✅ 最適合這首歌）
   太靠琴頸 → 聲音悶糊、缺乏清晰度
```

### 手型要點

1. **手腕微微拱起**，像手裡握著一顆雞蛋。
2. **手指自然彎曲**，指尖朝向弦。
3. **拇指(p)** 在食指(i)前方，兩者形成一個「十字」。
4. **手腕保持放鬆**，不要僵硬。

## 2. 琶音（Arpeggio）

### 什麼是琶音？

琶音就是把和弦裡的音**一個一個分開彈**，而不是一次刷完全部。

```
 和弦刷弦（Strumming）       vs.       琶音（Arpeggio）
   ┃ ┃ ┃ ┃ ┃ ┃                        ┃
   ┃ ┃ ┃ ┃ ┃ ┃ ← 同時發聲            ┃    ← 一個一個發聲
   ↓ ↓ ↓ ↓ ↓ ↓                        ┃ ┃
   嘩～（一聲）                         ┃ ┃ ┃
                                        ┃ ┃ ┃ ┃
                                        叮～叮～叮～叮～
```

在這首歌中，**前奏**和**主歌（Verse）** 使用琶音，營造**安靜、私密、訴說**的氛圍。

---

### 2.1 基本琶音模式（模式 A）

**適用段落：** Intro、Verse
**節奏：** 每小節（4 拍）彈 8 個音（每拍 2 個音 = 八分音符）

```
拍子：   1       &       2       &       3       &       4       &
右手：   p       i       m       a       p       i       m       a
彈弦：  根音弦   3弦     2弦     1弦    根音弦   3弦     2弦     1弦

節奏記法（用 x 表示每個音出現的位置）：

拍子：   | 1   &   2   &   | 3   &   4   &   |
拇指(p)：| x           |   | x               |
食指(i)：|     x       |   |     x           |
中指(m)：|         x   |   |         x       |
無名指(a)|           x |   |             x   |
```

### 各和弦的根音弦位置

拇指(p)需要根據不同和弦去彈不同的低音弦，這是琶音中最需要注意的部分：

| 和弦      | p（拇指）彈哪根弦 | 原因                       |
| --------- | ----------------- | -------------------------- |
| G/B       | **5 弦**          | B 音在 5 弦第 2 格         |
| Cadd9 / C | **5 弦**          | C 音（根音）在 5 弦第 3 格 |
| D         | **4 弦**          | D 音（根音）在 4 弦空弦    |
| Em        | **6 弦**          | E 音（根音）在 6 弦空弦    |
| Am        | **5 弦**          | A 音（根音）在 5 弦空弦    |
| G         | **6 弦**          | G 音（根音）在 6 弦第 3 格 |

> [!IMPORTANT]
> **拇指的目標是「唱出低音旋律」。** 低音線走得好，整個琶音就會很有根基，像一棟房子的地基一樣穩固。

---

### 2.2 用 TAB 譜理解：Verse 的琶音

以 Verse 進行 `G/B → Cadd9 → D → Em` 為例：

```
   G/B（一小節）                   Cadd9（一小節）
e|----3-------3----------|----0-------0----------|
B|------0-------0--------|------3-------3--------|
G|--------0-------0------|--------0-------0------|
D|                       |--2-------2------------|
A|--2-------2------------|--3-------3------------|
E|                       |                       |

   D（一小節）                     Em（一小節）
e|----2-------2----------|----0-------0----------|
B|------3-------3--------|------0-------0--------|
G|--------2-------2------|--------0-------0------|
D|--0-------0------------|--2-------2------------|
A|                       |--2-------2------------|
E|                       |--0-------0------------|
```

**怎麼讀 TAB 譜？**

```
e|----3-------     ← 第 1 弦，彈第 3 格的位置
B|------0-----     ← 第 2 弦，彈空弦（0 = 空弦）
G|--------0---     ← 第 3 弦，彈空弦
D|
A|--2---------     ← 第 5 弦，彈第 2 格
E|

   ↑               ← 時間從左往右流動
   先彈             ← 最左邊的音先彈
```

- 數字 = 彈在哪一格（0 = 空弦）
- 六條線 = 吉他的六根弦（上面 = 細弦，下面 = 粗弦）
- 從左到右 = 時間順序

---

### 2.3 進階琶音模式（模式 B）

當基本模式已經穩定後，可以嘗試「回頭」型琶音：

```
拍子：   1       &       2       &       3       &       4       &
右手：   p       i       m       i       p       i       m       a
彈弦：  根音弦   3弦     2弦     3弦    根音弦   3弦     2弦     1弦
                         ↑
                    回到 3 弦（「回頭」）
```

對比：

| 模式   | 節奏感                      | 效果                   |
| ------ | --------------------------- | ---------------------- |
| 模式 A | p-i-m-a-p-i-m-a（直線上行） | 規律、穩定             |
| 模式 B | p-i-m-i-p-i-m-a（中間回頭） | 多了一點「搖曳」的感覺 |

---

### 2.4 琶音練習步驟

從最簡單開始，分四階段：

**第一階段：不按左手**

```
右手只練 p-i-m-a 的動作
全部彈空弦
專注在手指的穩定性和音量的均勻
節拍器：60 BPM
```

**第二階段：按固定和弦**

```
左手按住 Em（最簡單的和弦）
右手練 p-i-m-a
確認每個音都清脆
節拍器：60 BPM
```

**第三階段：兩和弦切換**

```
| Em（4拍）| Am（4拍）| → 重複
右手持續 p-i-m-a 不停
左手在拍子上準時換和弦
節拍器：60 BPM → 慢慢加到 80 BPM
```

**第四階段：完整 Verse 進行**

```
| G/B（4拍）| Cadd9（4拍）| D（4拍）| Em（4拍）| → 重複
節拍器：60 BPM → 慢慢加到 103 BPM
```

## 3. 刷弦（Strumming）

### 什麼時候用刷弦？

在這首歌中，刷弦出現在**情緒開始升溫**的段落：

| 段落                   | 彈法            |
| ---------------------- | --------------- |
| Pre-Chorus（預備副歌） | 琶音 → 刷弦過渡 |
| Chorus（副歌）         | 全刷弦          |
| Bridge（橋段）         | 全刷弦（最強）  |

### 3.1 刷弦的正確手型

```
刷弦可以用以下方式：

方式 1: 用食指指甲的側面刷（最常見）
   → 手指略微彎曲，食指帶領其他手指自然跟隨

方式 2: 用 Pick（撥片）
   → 握在拇指和食指之間，露出約 3-5mm

這首歌的風格比較溫柔，建議用「手指刷弦」而非 Pick，
音色會更柔和、更貼近原曲。
```

### 重要原則

1. **動力來自手腕** —— 不是手臂、不是手指，是手腕的旋轉。
2. **手腕要放鬆** —— 像在甩水一樣自然。
3. **力度均勻** —— 除非刻意做重音，否則每下刷弦的力度要一致。

---

### 3.2 推薦刷弦型態（Strumming Pattern）

**這首歌使用的主要節奏型：**

```
拍子：   1       &       2       &       3       &       4       &
動作：   ↓       .       ↓       ↑       .       ↑       ↓       ↑
記法：   D       -       D       U       -       U       D       U

符號說明：
  ↓ / D = 下刷（Down Stroke）── 從粗弦往細弦方向刷
  ↑ / U = 上刷（Up Stroke）  ── 從細弦往粗弦方向刷
  .  / - = 手腕做動作但不碰弦（Ghost Stroke，空刷）
```

### 3.3 理解「空刷」的關鍵概念

很多初學者對 `D - D U - U D U` 中的 `-`（空刷）感到困惑。
這裡有一個核心觀念要理解：

> **手腕的上下擺動是永遠不停的。**

```
拍子：  1    &    2    &    3    &    4    &
手腕：  ↓    ↑    ↓    ↑    ↓    ↑    ↓    ↑   ← 手腕永遠在擺
碰弦：  ✓    ✗    ✓    ✓    ✗    ✓    ✓    ✓   ← 有些拍刻意「閃過」不碰弦
結果：  D    -    D    U    -    U    D    U
```

手腕就像一個鐘擺，永遠在 ↓↑↓↑ 地擺動。你只是選擇「哪一下碰弦、哪一下閃過」。這樣思考，節奏就不再神秘了。

### 3.4 刷弦練習步驟

**第一階段：手腕擺動（不碰弦）**

```
節拍器：60 BPM
手腕跟著拍子 ↓↑↓↑↓↑↓↑ 地擺動
嘴巴數：「1 and 2 and 3 and 4 and」
持續 1 分鐘
```

**第二階段：全下刷**

```
節拍器：60 BPM
每拍一下下刷：↓ ↓ ↓ ↓
左手按固定和弦（如 Em 或 Am）
持續 1 分鐘
```

**第三階段：加入上刷**

```
節拍器：60 BPM
每拍 ↓↑：↓↑ ↓↑ ↓↑ ↓↑
所有 & 拍都上刷
持續 1 分鐘
```

**第四階段：加入空刷**

```
節拍器：60 BPM
D - D U - U D U
↓ . ↓ ↑ . ↑ ↓ ↑
嘴巴數：「DOWN - DOWN UP - UP DOWN UP」
持續 2 分鐘
```

**第五階段：加入和弦切換**

```
節拍器：60 BPM → 逐步加到 103 BPM
| Am（1小節）| C（1小節）| G（1小節）| D（1小節）| → 重複
```

---

### 3.5 極簡替代刷法（如果節奏型太難）

完全不需要覺得丟臉，先用最簡單的方式把歌曲結構學會：

```
拍子：  1    2    3    4
動作：  ↓    ↓    ↓    ↓

每拍一下下刷。先專注在「和弦轉換的時機點」。
等和弦轉換流暢了，再替換成正式的節奏型。
```

## 4. 從琶音到刷弦的「過渡」

這是這首歌最精妙的編排之一：Pre-Chorus 段落中，伴奏從琶音**逐漸轉化**為刷弦，配合情緒的升溫。

### 過渡發生在哪裡？

```
Pre-Chorus 和弦進行：
| Am     | C      | G      | D      |

彈法分配：
| Am     | C      | G      | D      |
  琶音     琶音    開始刷弦   完全刷弦
(力度稍強)                   (為副歌準備)
```

### 具體過渡手法

**第 1-2 小節（Am → C）：加強版琶音**

```
還是 p-i-m-a 的琶音，但是：
- 力度比 Verse 稍強
- 可以讓每個音稍微持續更長（Let Ring）
- 音量漸漸加大
```

**第 3 小節（G）：開始掃弦過渡**

```
前半小節用琶音的尾巴，後半小節開始輕刷：

拍子：  1    &    2    &    3    &    4    &
方式：  p    i    m    a    ↓    .    ↓    ↑
        ← 琶音 →         ← 開始刷弦 →
```

**第 4 小節（D）：正式刷弦**

```
D - D U - U D U
進入完整的刷弦節奏型
```

> [!TIP]
> **過渡不需要很精確。** 重點是「自然」。想像你在說話，前面的部分是喃喃低語，後面的部分語氣逐漸加重 —— 不需要一個精確的切換點，讓它發生就好。

## 5. 力度控制

### 琶音的力度控制

```
pp（非常輕）：  指尖幾乎只是「觸碰」弦面    → Intro 前奏
p （輕）    ：  正常力度撥弦                  → Verse 主歌
mp（中弱）  ：  稍微用力一點                  → Pre-Chorus 前半
```

### 刷弦的力度控制

```
mf（中強）  ：  手腕擺幅中等                  → Pre-Chorus 後半
f （強）    ：  手腕擺幅加大，碰弦面積更多    → Chorus 副歌
ff（非常強）：  手腕全力擺動，每下都飽滿有力  → Bridge 橋段
```

### 實用技巧

- **要變大聲** → 手腕擺動幅度加大、碰弦的面積增加。不是「更用力地抓弦」。
- **要變小聲** → 手腕擺動幅度縮小，碰弦面積減少。
- **重音** → 在特定拍子上稍微加大幅度（通常是第 1 拍和第 3 拍）。
