# 07 — 練習計畫與常見問題

> 比「知道怎麼彈」更重要的，是「知道怎麼練」。

---

## 本篇目標

- ✅ 獲得一份結構化的三週練習計畫
- ✅ 建立正確的練習觀念
- ✅ 解決學習過程中最常遇到的疑問

---

## 1. 三週練習計畫

> [!NOTE]
> 這個時間表假設你**每天能練習 30 分鐘**。如果每天只有 15 分鐘，時程可以等比例拉長（變成六週）。重點不是多快完成，而是**每一步都扎實**。

---

### 📅 第一週：建造地基

**目標：** 所有和弦都按得出來、琶音能穩定地彈。

| 天數  | 練習項目   | 細節                                                                         | 時間分配 |
| ----- | ---------- | ---------------------------------------------------------------------------- | -------- |
| Day 1 | 認識和弦   | 對照 [03 篇](03_和弦全攻略.md)，逐一練習按壓每個和弦，進行「單弦清晰度測試」 | 30 分鐘  |
| Day 2 | 和弦微調   | 繼續清晰度測試，找出按不乾淨的弦並修正手指角度                               | 20 分鐘  |
|       | 右手暖身   | 不按和弦，右手練空弦 p-i-m-a 琶音                                            | 10 分鐘  |
| Day 3 | 兩和弦切換 | Em↔Am、Am↔C 各練 3 分鐘（1 分鐘計時法）                                      | 15 分鐘  |
|       | 琶音練習   | Em 和弦 + p-i-m-a 琶音，60 BPM                                               | 15 分鐘  |
| Day 4 | 兩和弦切換 | G/B↔Cadd9、D↔Em 各練 3 分鐘                                                  | 15 分鐘  |
|       | 琶音練習   | G/B + Cadd9 兩和弦循環，琶音，60 BPM                                         | 15 分鐘  |
| Day 5 | 兩和弦切換 | Cadd9↔D、C↔G（最難的兩組）                                                   | 15 分鐘  |
|       | 琶音練習   | 四和弦循環 G/B→Cadd9→D→Em，琶音，60 BPM                                      | 15 分鐘  |
| Day 6 | 四和弦循環 | Verse 進行琶音，70 BPM                                                       | 20 分鐘  |
|       | 刷弦入門   | 固定一個和弦，練基本下刷 ↓↓↓↓                                                | 10 分鐘  |
| Day 7 | 總複習     | Verse 琶音 + Pre-Chorus 和弦（先用下刷），80 BPM                             | 30 分鐘  |

**第一週結束時的自我評估：**

- [ ] 每個和弦通過「單弦清晰度測試」？
- [ ] 四和弦琶音循環能以 70 BPM 彈 2 分鐘不中斷？
- [ ] 最難的和弦轉換（Cadd9→D）能以每分鐘 20 次以上？

### 📅 第二週：模塊組裝

**目標：** 把各段落單獨學會，練習刷弦節奏型。

| 天數  | 練習項目       | 細節                                                              | 時間分配 |
| ----- | -------------- | ----------------------------------------------------------------- | -------- |
| Day 1 | 前奏 + Verse 1 | 對照 [05 篇](05_分段教學篇.md)，琶音彈完前奏→Verse 1，80 BPM      | 20 分鐘  |
|       | 刷弦節奏型     | 練 D-D U-U D U，單和弦固定，60 BPM                                | 10 分鐘  |
| Day 2 | 刷弦節奏型     | D-D U-U D U + Am→C→G→D 和弦切換，60 BPM                           | 20 分鐘  |
|       | Pre-Chorus     | 練習琶音→刷弦的過渡（Am 琶音→C 琶音→G 刷弦→D 刷弦）               | 10 分鐘  |
| Day 3 | 副歌           | Chorus 的 12 小節，全刷弦，80 BPM                                 | 20 分鐘  |
|       | 琶音複習       | 前奏 + Verse 用 90 BPM 彈一遍                                     | 10 分鐘  |
| Day 4 | 串聯 A 面      | Intro → Verse 1 → Pre-Chorus 1 → Chorus 1，90 BPM                 | 30 分鐘  |
| Day 5 | 橋段 + 尾奏    | Bridge（ff 刷弦）+ Outro（琶音漸弱），80 BPM                      | 20 分鐘  |
|       | 補強弱點       | 回去練你覺得最卡的那個轉換或段落                                  | 10 分鐘  |
| Day 6 | 串聯 B 面      | Verse 2 → Pre-Chorus 2 → Chorus 2 → Bridge → Final Chorus → Outro | 30 分鐘  |
| Day 7 | 首次完整嘗試   | 從 Intro 到 Outro 一口氣彈完（允許犯錯、不停），90 BPM            | 30 分鐘  |

**第二週結束時的自我評估：**

- [ ] 每個段落單獨都能穩定彈完？
- [ ] 刷弦節奏型 D-D U-U D U 能以 80 BPM 穩定 1 分鐘？
- [ ] 嘗試過至少一次完整演奏（從頭到尾）？

### 📅 第三週：打磨細節

**目標：** 速度達標、力度控制到位、可以流暢演奏全曲。

| 天數  | 練習項目  | 細節                                      | 時間分配 |
| ----- | --------- | ----------------------------------------- | -------- |
| Day 1 | 加速練習  | 全曲完整彈，95 BPM                        | 30 分鐘  |
| Day 2 | 力度控制  | 特別練習段落間的力度對比（pp↔f↔ff）       | 20 分鐘  |
|       | 加速練習  | 全曲 100 BPM                              | 10 分鐘  |
| Day 3 | 達標速度  | 全曲 103 BPM（原速）                      | 20 分鐘  |
|       | 錄音自聽  | 用手機錄下來，聽一遍找問題                | 10 分鐘  |

- [ ] 能以 103 BPM 完整彈完全曲？

- [ ] 錄音聽起來跟原曲「同一首歌」？

## 2. 練習的核心觀念

### 觀念 1：慢就是快

```
     新手的直覺           應該做的
      ↓                    ↓
  「我要趕快彈到原速！」  「我要先彈對、彈穩。」
      ↓                    ↓
  用 103 BPM 練習         用 60 BPM 練習
      ↓                    ↓
  到處犯錯、手忙腳亂      每個音清清楚楚
      ↓                    ↓
  反覆練錯的東西           建立正確的肌肉記憶
      ↓                    ↓
  練了一週還是亂的 😩      一週後穩定加速到 103 😎
```

> **你的肌肉不分對錯 —— 你練什麼，它就記什麼。**
> 如果在 103 BPM 下練錯了 100 遍，你的手指就會「記住錯誤的路徑」，之後要花更多時間才能糾正。

### 觀念 2：分段 > 從頭到尾

不要每次都從前奏開始彈。如果你最弱的段落是 Pre-Chorus 的過渡，那就**單獨把那 4 小節拿出來**練 50 遍。

```
✅ 有效率的練習：
   「今天我要征服 Cadd9 → D 的轉換」
   → 只練這一個轉換 15 分鐘

❌ 沒效率的練習：
   「我從頭彈到尾」
   → 前奏和 Verse 已經很熟了，白白花時間
   → 到 Cadd9→D 時犯錯，但已經練了 20 分鐘很累了
```

### 觀念 3：每天 30 分鐘 > 一週一次 3 小時

```
每天 30 分鐘 × 7 天 = 7 次「入睡→起床→肌肉記憶鞏固」的循環
一週一次 3 小時     = 1 次循環

      哪個效果好？           前者。壓倒性地好。
```

人的肌肉記憶形成需要**睡眠**。每次睡覺時，大腦會把當天練習的動作「存檔」。所以練 7 天 = 存檔 7 次，遠勝於一次練到手指痛然後存檔 1 次。

### 觀念 4：錄音是最好的老師

你自己彈的時候，大腦忙著處理「按什麼和弦、節奏怎麼走」，沒有餘力去「客觀地聽」。

錄下來回放，你會馬上發現：

- 「原來這裡節奏不穩」
- 「原來力度一直很平，沒有變化」
- 「原來 Cadd9 的第 2 弦有雜音」

用手機錄就夠了，不需要專業設備。

## 3. 常見問題 Q&A

### ❓ Q1：我不會日文，歌詞怎麼唱？

你有三個選擇：

| 方法         | 說明                                    | 適合誰                   |
| ------------ | --------------------------------------- | ------------------------ |
| **羅馬拼音** | 在 YouTube 搜尋「晩餐歌 歌詞 ローマ字」 | 想唱原版日文的人         |
| **只彈不唱** | 純演奏，專注在吉他表現                  | 不想唱歌的人             |
| **哼旋律**   | 用「啦」或「嗯」跟著旋律哼              | 想有歌聲但不想記歌詞的人 |

> 建議先用第三種方式（哼旋律）把「邊彈邊發出聲音」的協調性練好，再決定要不要挑戰日文歌詞。

---

### ❓ Q2：Cadd9 和 C 到底用哪個？

**簡短回答：** 兩個都可以，差別在於音色。

| 和弦  | 組成音         | 音色特點                 |
| ----- | -------------- | ------------------------ |
| C     | C-E-G + B(2弦) | 穩定、完整、傳統         |
| Cadd9 | C-E-G + D(2弦) | 空靈、懸浮、帶有未完成感 |

Cadd9 的第 2 弦（D 音）和第 1 弦（E 空弦）之間只差一個全音，靠得很近，會產生一種微妙的「摩擦感」—— 這就是它聽起來比較「空靈」的原因。

**結論：** 如果小指夠靈活 → 用 Cadd9；如果覺得按不到 → 先用 C，完全不影響學習。

---

### ❓ Q3：G/B 和弦為什麼不用普通 G？

詳細的音樂理論在 [03 篇](03_和弦全攻略.md) 已經解釋過了。簡單來說：

**G/B 讓低音走出一條上行旋律線：B → C → D → E**

這條線把四個和弦串在一起，聽起來特別流暢。如果用普通 G，低音就會跳來跳去（G → C → D → E），少了那份「連貫感」。

**但如果暫時按不好 G/B → 先用 G，後面再換回來。**

---

### ❓ Q4：琶音的手指不夠靈活怎麼辦？

這是最常見的問題之一，原因通常是以下三點：

| 原因         | 症狀                   | 解法                           |
| ------------ | ---------------------- | ------------------------------ |
| 手腕太僵硬   | 手指勉強在動，但很費力 | 放鬆手腕！動作來自指根關節     |
| 速度太快     | 音符含糊不清           | 降速到你能完美彈出的 BPM       |
| 手指分配錯誤 | i-m-a 搶弦或漏弦       | 每根手指要習慣「只負責一根弦」 |

**最有效的練法：**

```
1. 不按左手和弦，右手只練空弦 p-i-m-a
2. 節拍器設 40 BPM（是的，非常慢）
3. 每個音都要「聽到自己彈出來」
4. 持續 5 分鐘後加到 50 BPM
5. 每天這樣練一次，一週後會大幅改善
```

---

### ❓ Q5：從琶音過渡到刷弦很突兀怎麼辦？

分三步實現**自然過渡**：

```
第 1 步：音量漸強
   └─ 琶音的力度逐漸加大（pp → p → mp → mf）

第 2 步：最後幾拍同時勾弦
   └─ 原本 p-i-m-a 一個一個彈，改成 i-m-a 三根手指同時彈
   └─ 這樣聽起來就接近「輕刷」了

第 3 步：自然進入刷弦
   └─ 從同時勾弦 → 改成手腕帶動掃弦
   └─ 過渡完成！
```

> 關鍵：不要想著「現在要從琶音切換成刷弦」，而是想著「我的聲音要越來越大」。力度的增加會自然帶動你從琶音轉向刷弦。

### ❓ Q6：有沒有更簡單的方法彈這首歌？

**有！以下是三個難度等級：**

#### 極簡版（★☆☆☆☆）

```
和弦：只用 Em、C、G、Am（全部是最基本的開放和弦）
彈法：每個和弦下刷 4 次（↓ ↓ ↓ ↓）
節奏：不需要刻意分琶音和刷弦

適合：拿起吉他不到一個月的完全初學者
```

#### 簡化版（★★☆☆☆）

```
和弦：C（代替 Cadd9）、G（代替 G/B）、D、Em、Am
彈法：全曲用下刷 ↓ ↓ ↓ ↓ 或 ↓↓↑↑↓↑
節奏：不分琶音和刷弦，統一刷弦

適合：有基本和弦基礎但還沒練過琶音的人
```

#### 完整版（★★★☆☆）= 本系列教的版本

```
和弦：G/B、Cadd9、D、Em、Am、C、G
彈法：Verse 琶音 + Chorus 刷弦 + 力度變化
節奏：D - D U - U D U

適合：有一點基礎、願意花三週好好練的人
```

### ❓ Q7：手指很痛，正常嗎？

**完全正常。** 這是每個吉他初學者都會經歷的過程。

你的指尖需要時間長出**繭**（Callus），這是一層硬化的皮膚，長出來之後按弦就不痛了。

| 時間點    | 感受                           |
| --------- | ------------------------------ |
| 第 1-3 天 | 手指尖很痛，甚至紅腫           |
| 第 4-7 天 | 還是痛，但開始習慣了           |
| 第 2 週   | 摸起來有硬硬的感覺（繭在形成） |
| 第 3-4 週 | 大多數情況不太痛了             |
| 1 個月後  | 幾乎不痛，繭完全形成           |

**注意事項：**

- 如果只是指尖疼痛 → 正常，休息一下再練。
- 如果手腕、前臂、或手指關節痛 → **停下來！** 這可能是姿勢不正確造成的，需要調整。
- 每天練習時間不要超過 1 小時（初學者），給手指恢復時間。
