# 🎸 GHOST — 電吉他教學系列

> 星街すいせい（Hoshimachi Suisei）的搖滾名曲，從零開始征服它。

---

## 這個系列是什麼？

這是一份為**電吉他初中級～中級玩家**設計的系統化教學。「GHOST」不是一首單純的流行歌 —— 它有紮實的搖滾吉他編曲、吉他 Solo、強力和弦（Power Chord）、悶音（Palm Mute）等技巧。我們會把這些拆解成可消化的模塊，一塊一塊學會。

## 為什麼選這首歌？

| 特點             | 說明                                                  |
| ---------------- | ----------------------------------------------------- |
| **搖滾基本功**   | Power Chord、Palm Mute、Distortion —— 電吉他必備技巧  |
| **有 Solo 段落** | 學會你的第一段搖滾吉他 Solo                           |
| **結構豐富**     | Intro → Verse → Chorus → Solo → Bridge → 完整搖滾結構 |
| **動態極端**     | 從安靜的 Verse 到爆發的 Chorus，練習力度控制          |
| **BPM 適中**     | 91 BPM（體感 182 的八分音符律動），不算太快           |

> [!IMPORTANT]
> **這首歌比「晩餐歌」難度更高。** 如果你是電吉他的完全新手，建議先有以下基礎再開始：
>
> - 能按住 Power Chord 並穩定刷弦
> - 能使用 Pick（撥片）上下交替撥弦
> - 知道什麼是 Palm Mute（悶音）
>
> 如果這些你都還不太熟，不用擔心 —— 本系列會從頭教起，只是進度會比較慢一些。
