# 01 — 認識這首歌

> 在插上音箱之前，先讓大腦「聽懂」這首歌。

## 1. 歌曲基本資訊

| 項目     | 內容                                      |
| -------- | ----------------------------------------- |
| 歌名     | GHOST                                     |
| 歌手     | 星街すいせい（Hoshimachi Suisei）         |
| 作詞     | 星街すいせい                              |

### 關於這首歌

「GHOST」是 VTuber 星街すいせい的代表作之一，由她本人作詞。歌詞描述一個像幽靈一樣不被看見的存在，拼命想讓自己的聲音被聽見、被證明。這首歌帶有很強的搖滾基因 —— 重拍的鼓組、有力的電吉他 Riff、漸進式的情緒爆發，以及一段令人熱血的吉他 Solo。

對電吉他手來說，這首歌是一個非常好的**綜合訓練場**：它幾乎涵蓋了搖滾吉他的所有基本功。

## 2. 調性分析

### 原調

| 概念     | 數值                            |
| -------- | ------------------------------- |
| 原調     | F# 大調 / G♭ 大調（等音調）     |
| 調弦方式 | **半音下降調弦（E♭ Standard）** |

### 什麼是半音下降調弦？

標準調弦是 E A D G B E。半音下降調弦就是把每根弦都**降低半音**：

```
標準調弦：     E    A    D    G    B    E
                ↓    ↓    ↓    ↓    ↓    ↓  （每根弦降半音）
半音下降調弦：E♭   A♭   D♭   G♭   B♭   E♭
```

### 為什麼要用半音下降調弦？

1. **音色更暗、更厚重。** 弦的張力降低，振動幅度更大，產生更飽滿、更「沉」的音色，非常適合搖滾和另類搖滾風格。

2. **人聲配合。** 歌手的音域有時候在兩個調之間，降半音可以讓歌手更舒適地演唱。

3. **搖滾傳統。** 從 Jimi Hendrix、Guns N' Roses 到 Nirvana，無數經典搖滾樂團都使用半音下降調弦。

> [!NOTE]
> **如果你不想調弦也可以。** 用標準調弦彈完全一樣的指型，聲音只是比原曲高半音。在練習階段完全不影響，等到要跟原曲合奏時再調就好。

## 3. 節拍與速度

| 項目     | 數值                               |
| -------- | ---------------------------------- |
| 拍號     | 4/4 拍                             |
| BPM      | 91（基本速度）                     |
| 體感速度 | 182 BPM（八分音符律動）            |
| 風格     | 中速搖滾，帶有強烈的八分音符推進感 |

### 什麼是「體感 182 BPM」？

這首歌的基本速度是 91 BPM（每分鐘 91 個四分音符），但在實際演奏中，吉他和鼓的律動是以**八分音符**為基本單位的。也就是說，你的手（Pick）一秒鐘要上下擊弦大約 3 次，體感就像 182 BPM 一樣快。

```
4/4 拍，91 BPM：

四分音符：  |  1        2        3        4        |
八分音符：  |  1   &    2   &    3   &    4   &    |
             ↓ ↑   ↓ ↑   ↓ ↑   ↓ ↑   ↓ ↑   ↓ ↑   ↓ ↑   ↓ ↑
             └─────── 這才是你右手實際擊弦的頻率 ───────┘
```

## 4. 歌曲結構圖

「GHOST」的結構比一般流行歌更豐富，包含 **11 個段落**，是典型的搖滾歌曲結構。

### 段落名稱對照

| 英文名       | 中文名    | 功能                               |
| ------------ | --------- | ---------------------------------- |
| **Intro**    | 前奏      | 電吉他 Riff 建立歌曲的「聲音招牌」 |
| **Verse A**  | 主歌 A    | 安靜的敘事段，Clean 或 Crunch 音色 |
| **Chorus A** | 副歌 A    | 第一次爆發，Distortion 全開        |
| **Hook**     | 過門      | 短小精幹的連接段，回到 Riff        |
| **Verse B**  | 主歌 B    | 第二段主歌，力度可能比 A 稍強      |
| **Chorus B** | 副歌 B    | 第二次爆發，比第一次更猛烈         |
| **Pre-Solo** | Solo 預備 | 為吉他 Solo 做鋪墊                 |
| **Solo**     | 吉他獨奏  | 全曲的技術高峰                     |
| **Bridge**   | 橋段      | 情緒轉折，通常有新的和弦或旋律     |
| **Chorus C** | 最終副歌  | 全力爆發的最後一次副歌             |
| **Outro**    | 尾奏      | 收尾，通常回到 Intro 的 Riff       |

### 完整結構時間軸

```
 ┌───────┐ ┌─────────┐ ┌──────────┐ ┌──────┐ ┌─────────┐
 │ Intro │→│ Verse A │→│ Chorus A │→│ Hook │→│ Verse B │→
 │ Riff  │ │  Clean  │ │Distortion│ │ Riff │ │Clean/Cr.│
 │ mf    │ │   p     │ │   f      │ │  mf  │ │  p→mp   │
 └───────┘ └─────────┘ └──────────┘ └──────┘ └─────────┘

 ┌──────────┐ ┌──────────┐ ┌──────┐ ┌─────────┐
→│ Chorus B │→│ Pre-Solo │→│ Solo │→│ Bridge  │→
 │Distortion│ │  Build   │ │ Lead │ │ Rebuild │
 │   f      │ │  mp→f    │ │  ff  │ │  p→f    │
 └──────────┘ └──────────┘ └──────┘ └─────────┘

 ┌──────────┐ ┌───────┐
→│ Chorus C │→│ Outro │
 │Distortion│ │ Riff  │
```

## 5. 吉他在這首歌中的角色

「GHOST」的吉他編曲至少分為**兩軌**（Rhythm Guitar + Lead Guitar）。在自己練習時，我們主要學習 **Rhythm Guitar（節奏吉他）** 的部分，Solo 段落則學習 Lead Guitar。

### 各段落吉他角色

| 段落   | 音色            | 做什麼                        |
| ------ | --------------- | ----------------------------- |
| Intro  | Crunch/Dist.    | 彈主 Riff（招牌旋律）         |
| Verse  | Clean/Crunch    | 低調的和弦或琶音，配合歌聲    |
| Chorus | Distortion      | 全力 Power Chord 刷弦         |
| Hook   | Crunch/Dist.    | 回到 Intro 的 Riff            |
| Solo   | Distortion+Lead | 旋律性的 Solo（需要較高技巧） |
| Bridge | Clean→Dist.     | 從安靜重新建立到爆發          |
| Outro  | Distortion      | Riff 重現後漸弱               |

```
    │ In  V.A Ch.A Hk V.B Ch.B Pre Sol Bridge    Ch.C    Outro
```

### 解讀重點

1. **Verse 是「最低谷」** —— 安靜、內斂。吉他可能只是輕撥 Clean 音色的和弦。
2. **Chorus 是「浪頭」** —— 每次 Chorus 力度都比前一次更強。
3. **Solo 是「頂峰」** —— 技術與情感的雙重高潮。
4. **Bridge 有「先下後上」的弧線** —— 先沉下去再爬上來，為最後一次 Chorus 做跳板。
5. **Outro 逐漸消退** —— 像回聲一樣慢慢消失。

## 7. 核心和弦速查

這首歌用到的主要和弦（以 F# 大調為基礎）：

| 和弦 | 級數       | Power Chord 寫法 | 根音位置（6弦） | 根音位置（5弦） |
| ---- | ---------- | ---------------- | --------------- | --------------- |
| F#   | I（一級）  | F#5              | 2 格            | —               |
| B    | IV（四級） | B5               | 7 格            | 2 格            |
| C#   | V（五級）  | C#5              | 9 格            | 4 格            |
| D#m  | vi（六級） | D#5              | 11 格           | 6 格            |
| E    | ♭VII       | E5               | —               | 7 格            |

> Power Chord（強力和弦）只需要按兩個音（根音 + 5 度音），是搖滾吉他的核心。詳細教學見 [04 篇](04_Riff與和弦篇.md)。

## 8. 第一步行動：聽歌

### 🎧 至少完整聽三遍

1. **第一遍：感受整體** — 注意歌曲什麼時候「安靜」、什麼時候「爆炸」。
2. **第二遍：數段落** — 試著用手指打拍子，感受 91 BPM（或 182 的八分音符）的速度。同時嘗試聽出段落的切換點。
3. **第三遍：聽吉他** — 專注在電吉他上。什麼時候聽到乾淨的音色？什麼時候變成粗獷的破音？Intro 的 Riff 長什麼樣？Solo 在哪裡出現？

> [!TIP]
> 如果可以，戴上耳機。在混音中，吉他通常被分配在左右聲道（Pan L/R），用耳機可以更清楚地聽到吉他在做什麼。
