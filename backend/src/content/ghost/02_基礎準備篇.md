## 1. 裝備清單

### 必需品

| 裝備              | 說明                                                                        |
| ----------------- | --------------------------------------------------------------------------- |
| 🎸 電吉他         | 任何類型都可以（Strat、Tele、LP、SG...），有雙線圈拾音器（Humbucker）更理想 |
| 🔌 導線           | 吉他到音箱/效果器的連接線，至少一條（3～5 米）                              |
| 🎛️ 音箱 或 模擬器 | 可以是實體音箱、音箱模擬踏板、或電腦軟體（如 AmpliTube、BIAS FX）           |
| 🎵 調音器         | 夾式調音器 或 踏板式調音器 或 App                                           |
| 🔺 Pick           | 建議厚度 **0.73mm～1.0mm**（Medium 到 Heavy）                               |
| 🎧 耳機           | 練習時不想打擾鄰居的話                                                      |
| 📱 節拍器         | App 或 Google 線上版                                                        |

### 加分裝備

| 裝備            | 說明                                              |
| --------------- | ------------------------------------------------- |
| 🎛️ 破音效果器   | Overdrive 或 Distortion 踏板                      |
| 🎛️ Delay/Reverb | Solo 段落可以讓聲音更有空間感                     |
| 📱 DAW 軟體     | 錄音用（GarageBand、Reaper 等）                   |
| 🎵 減速 App     | Anytune / Amazing Slow Downer，可以把原曲放慢練習 |

## 2. 關於 Pick（撥片）的選擇

這首歌大量使用 **Palm Mute（悶音）** 和 **Power Chord 刷弦**，Pick 的選擇對手感影響很大：

| Pick 厚度    | 代號        | 適合         | 本曲建議 |
| ------------ | ----------- | ------------ | -------- |
| 0.46mm 以下  | Thin        | 民謠吉他刷弦 | ❌ 太軟  |
| 0.60～0.73mm | Medium      | 一般流行歌   | ⚠️ 可用  |
| 0.73～1.0mm  | Heavy       | 搖滾、金屬   | ✅ 推薦  |
| 1.0mm 以上   | Extra Heavy | 金屬、速彈   | ✅ 可用  |

> [!TIP]
> **推薦：0.88mm 左右的 Celluloid 或 Tortex 材質。** Dunlop Tortex 0.88mm（綠色）是搖滾吉他手的經典選擇。Pick 太薄會在 Palm Mute 時產生太多「啪啪聲」，太厚則不好控制刷弦。

## 3. 半音下降調弦（E♭ Standard / Half Step Down）

### 調弦對照表

```
         標準調弦              半音下降調弦
弦號   音名    頻率           音名     頻率
────   ────    ─────          ────     ─────
 6弦    E2     82.4 Hz   →   E♭2      77.8 Hz
 5弦    A2    110.0 Hz   →   A♭2     103.8 Hz
 4弦    D3    146.8 Hz   →   D♭3     138.6 Hz
 3弦    G3    196.0 Hz   →   G♭3     185.0 Hz
 2弦    B3    246.9 Hz   →   B♭3     233.1 Hz
 1弦    E4    329.6 Hz   →   E♭4     311.1 Hz
```

### 調弦步驟

1. 打開調音器。
2. 如果你的調音器支援「Flat Tuning」或「♭」模式，開啟它。這樣調音器顯示的字母還是 E、A、D、G、B、E，但每個都自動偏低半音。
3. 如果調音器沒有 Flat 模式，就手動把每根弦調到對應的降半音音名：

```
6 弦：調到 E♭ (Eb)  — 在 E 的基礎上往下轉，直到調音器顯示 Eb 正中央
5 弦：調到 A♭ (Ab)
4 弦：調到 D♭ (Db)
3 弦：調到 G♭ (Gb)
2 弦：調到 B♭ (Bb)
1 弦：調到 E♭ (Eb)
```

> [!WARNING]
> **調弦方向：全部要「放鬆」弦。** 半音下降是往低音方向調，所以你要**逆時針**旋轉弦鈕（讓弦變鬆）。如果不小心轉過頭了，要**先降得更低**，再慢慢往上調回來 —— 這樣弦的張力才會穩定，音準才不會飄。

### 不想重新調弦？

如果你不想來回切換調弦，以下方式同樣可以練習：

| 方法             | 做法                   | 優點           | 缺點                               |
| ---------------- | ---------------------- | -------------- | ---------------------------------- |
| 標準調弦直接彈   | 用一模一樣的指型       | 不用調弦       | 聲音比原曲高半音，跟原曲合奏會不合 |
| 移調彈奏         | 所有指型往高把位移一格 | 音高跟原曲一致 | 有些開放弦 Riff 會彈不出來         |
| 用數位效果器移調 | 用 Pitch Shift 降半音  | 什麼都不用改   | 音色可能稍有失真                   |

> **建議：先用標準調弦練技巧，等準備好要跟原曲合奏時再調成半音下降。**

## 4. 訊號鏈（Signal Chain）

電吉他的聲音從琴弦到你耳朵的路徑叫做**訊號鏈**。理解它有助於你調出想要的音色。

### 基本訊號鏈

```
🎸 電吉他
    │
    │ ← 導線
    ↓
🎛️ 效果器（Pedals）/ 模擬軟體
    │
    │ ← 導線
    ↓
🔊 音箱（Amp）
    │
    ↓
👂 你的耳朵
```

### 如果用電腦 / 模擬軟體

```
🎸 電吉他
    │
    │ ← 導線
    ↓
🎤 錄音介面（Audio Interface）
    │
    │ ← USB
    ↓
💻 電腦 + 音箱模擬軟體
    │           │ AmpliTube
    │           │ BIAS FX
    │           │ Guitar Rig
    │           │ Neural DSP
    ↓
🎧 耳機 / 喇叭
```

> [!NOTE]
> 如果你沒有音箱也沒有錄音介面，**插耳機到電吉他上是聽不到聲音的**（電吉他需要放大器）。最經濟的方案是買一個 USB 錄音介面（如 Focusrite Scarlett Solo）搭配免費的 AmpliTube 軟體。

## 5. 暖身運動

電吉他的暖身和木吉他類似，但需要特別針對 Pick 的控制和手指的速度做準備。

### 暖身 1：半音階爬格子（配合 Pick）

```
節拍器：60 BPM
Pick 方式：嚴格交替撥弦（Down-Up-Down-Up...）

TAB：

e|--------------------------------1--2--3--4--|
B|-------------------------1--2--3--4---------|
G|------------------1--2--3--4----------------|
D|-----------1--2--3--4-----------------------|
A|----1--2--3--4------------------------------|
E|--1--2--3--4--------------------------------|

上行完後反向下行：

e|--4--3--2--1--------------------------------|
B|-----------4--3--2--1-----------------------|
G|--------------------4--3--2--1--------------|
D|-----------------------------4--3--2--1-----|
A|------------------------------------4--3--2--1--|
E|-----------------------------------------4--3--2--1--|
```

**重點：**

- 每個音都必須清晰（沒有悶音、沒有碰弦雜音）
- Pick 嚴格 ↓↑↓↑ 交替，不要連續下撥
- 不追求速度，追求精準

### 暖身 2：Power Chord 快速移動

```
節拍器：80 BPM
在 6 弦上做 Power Chord 滑動：

TAB（6弦 + 5弦同時按）：

CHORD:  F#5    B5     C#5    B5     F#5
FRET:    2      7      9      7      2

E|-----------------------------------------------|
B|-----------------------------------------------|
G|-----------------------------------------------|
D|-----------------------------------------------|
A|--4------9------11-----9------4----------------|
E|--2------7------9------7------2----------------|
      ↓↓     ↓↓     ↓↓     ↓↓     ↓↓
```

每個和弦刷兩下（↓↓），然後快速滑到下一個位置。

### 暖身 3：Palm Mute 開關練習

```
節拍器：80 BPM
在一個 Power Chord（如 F#5，6弦第2格）上：

拍子：  1    &    2    &    3    &    4    &
狀態：  PM   PM   PM   PM   開   開   開   開
        悶   悶   悶   悶   放   放   放   放

PM = Palm Mute（手掌壓在琴橋附近悶住弦）
開 = 放開手掌，讓弦自由振動

重複練習「悶 → 開 → 悶 → 開」的切換
```
