# 03 — 音色與效果器篇

> 同樣的和弦、同樣的彈法，音色不對，味道就差了十萬八千里。

---

## 本篇目標

- ✅ 理解電吉他的三種基本音色（Clean / Crunch / Distortion）
- ✅ 知道這首歌的每個段落該用什麼音色
- ✅ 能用你手邊的器材調出接近原曲的聲音

---

## 1. 三種基本音色

### 音色頻譜圖

```
乾淨 ◄─────────────────────────────────────► 粗獷

  Clean          Crunch         Distortion
  │              │              │
  │ 透明、溫暖   │ 帶有磨砂感   │ 厚重、飽和
  │ 像原聲吉他   │ 微破不破     │ 轟隆隆的搖滾聲
  │ 刷弦很清晰   │ 有「咬感」   │ Power Chord 的主場
  │              │              │
  用在 Verse     用在 Intro     用在 Chorus + Solo
                  Hook          Bridge 爆發段
```

### 各音色詳解

#### Clean（乾淨音色）

```
Gain/Drive: 1～3 (低)
用途：Verse 的安靜段落
特點：每個音符都很清楚，沒有「嘶嘶」雜音
Tips：可以加一點 Reverb（殘響）讓聲音不那麼乾
```

#### Crunch（微破音色）

```
Gain/Drive: 4～6 (中)
用途：Intro Riff、Hook
特點：輕彈是 Clean，用力彈時會「破」，有動態感
Tips：這是這首歌吉他 Riff 的核心音色
      彈弱一點 = Clean，彈強一點 = 破
      非常適合表達歌曲中的力度變化
```

#### Distortion（全破音色）

```
Gain/Drive: 6～8 (高)
用途：Chorus、Solo、Bridge 爆發段
特點：聲音厚實飽滿，延音（Sustain）很長
Tips：Gain 不要開太高！開太高會失去清晰度，變成一團糊
      搖滾和 J-Rock 的甜蜜點通常在 6～7 左右
```

## 2. 這首歌的音色配置

| 段落         | 主音色           | Gain 建議 | 附加效果       |
| ------------ | ---------------- | --------- | -------------- |
| Intro        | Crunch           | 4～5      | 少許 Delay     |
| Verse A/B    | Clean            | 1～2      | 少許 Reverb    |
| Chorus A/B/C | Distortion       | 6～7      | —              |
| Hook         | Crunch           | 4～5      | —              |
| Pre-Solo     | Crunch→Dist.     | 5→7       | —              |
| Solo         | Distortion       | 7～8      | Delay + Reverb |
| Bridge       | Clean→Dist.      | 2→7       | Reverb         |
| Outro        | Distortion→Clean | 7→2       | Reverb（漸強） |

### 音色切換時機

```
時間軸 ─────────────────────────────────────────────────→

 Clean ：     ████████                    ████████
 Crunch：████          ████                        ████
 Dist. ：          ████     ████████████████
                                    ↑
         Intro Verse  Cho Hook Verse Cho PreS Solo Bri Cho Outro
```

> [!TIP]
> **如果你只有兩個音色可選**（例如用音箱的 Clean/Overdrive 切換），那就：
>
> - Clean 通道 → Verse、Bridge 前半
> - Overdrive 通道 → 其他所有段落
>
> 用**右手的力度**來模擬 Crunch 和 Distortion 的差別。

## 3. 各種器材的設定參考

### 方案 A：音箱模擬軟體（AmpliTube / BIAS FX / Neural DSP）

```
推薦音箱模型（Amp Model）：
  ├─ Clean：  Fender Twin Reverb 或 Roland JC-120
  ├─ Crunch： Marshall JCM800 (Gain 4-5)
  └─ Dist.：  Marshall JCM800 (Gain 7) 或 Mesa/Boogie Rectifier

EQ 參考：
  Bass:   5～6
  Mid:    6～7  ← 日式搖滾通常 Mid 不砍太多
  Treble: 5～6
  Presence: 5

效果器：
  ├─ Reverb: Room 或 Plate，Mix 20～30%
  ├─ Delay:  Tempo Sync 到 91 BPM，1/8 音符，Mix 15～20%
  └─ Noise Gate: 開啟（去除高 Gain 下的雜音）
```

### 方案 B：實體音箱 + 效果踏板

```
基本設定（以常見的 Boss Katana 或 Fender Mustang 為例）：

1. Clean 通道：
   Gain: 2  |  EQ 各中間  |  加少許 Reverb

2. Crunch/Dist. 設定：
   用一顆 Overdrive 踏板（如 Boss SD-1 或 Tube Screamer）
   Gain: 中等  |  Level: 高  |  Tone: 中偏高

3. 或直接用音箱的 Lead 通道：
   Gain: 6～7  |  EQ 同上

切換方式：用踏板的「ON/OFF」切換 Clean ↔ Overdrive
```

### 方案 C：什麼效果器都沒有

```
用電吉他上的 Volume（音量）旋鈕來控制：

- Verse：把 Volume 旋鈕轉到 3～4  → Clean 感覺
- Chorus：把 Volume 旋鈕轉到 10   → 推破音箱 → Distortion 感覺

這是很多職業吉他手的技巧！
音箱的 Gain 設在 5 左右，剩下的用吉他上的 Volume 控制。
```

## 4. 拾音器選擇（Pickup Selector）

如果你的電吉他有拾音器切換開關：

| 位置           | 音色特點             | 建議用在               |
| -------------- | -------------------- | ---------------------- |
| 琴頸（Neck）   | 溫暖、圓潤、柔和     | Clean 段落、Solo 旋律  |
| 中間（Middle） | 介於兩者之間         | 通用                   |
| 琴橋（Bridge） | 明亮、銳利、有攻擊性 | Riff、Power Chord 刷弦 |

```
這首歌的建議：

  Verse：琴頸拾音器（Neck） — 溫暖的 Clean 聲
  Riff / Chorus：琴橋拾音器（Bridge） — 有力的 Crunch/Dist.
  Solo：可以用琴頸 — 讓旋律音色更滑順
        也可以用琴橋 — 讓音色更亮、穿透力更強
        (兩種都可以，看你的喜好)
```

## 5. Noise Gate（噪音門）

電吉他在高 Gain 下會產生很多「嘶嘶」的雜音。**Noise Gate** 是一個效果，當你沒有彈奏的時候，它會自動把音量關掉，消除雜音。

```
設定建議：

  Threshold（閾值）：-40dB 左右（根據你的雜音大小調整）
  太低 → 雜音還是很大
  太高 → 你在彈的時候尾音會被截掉

如果你有模擬軟體，通常在效果鏈的最前面放一個 Noise Gate 就好。
如果是實體踏板，放在訊號鏈的最前面或最後面。
```

> [!WARNING]
> **Noise Gate 不是萬能的。** 如果雜音大到蓋過你的彈奏，通常是因為 Gain 開太高、或者接地不良。先降 Gain、檢查導線，再依賴 Noise Gate。
