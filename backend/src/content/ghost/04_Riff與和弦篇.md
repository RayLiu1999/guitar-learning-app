## 1. Power Chord（強力和弦）

### 什麼是 Power Chord？

Power Chord 是搖滾音樂中使用頻率最高的和弦形式。它只有**兩個不同的音**：

```
普通和弦（如 F# Major）：  F# + A# + C#  = 根音 + 3度 + 5度  (三個音)
Power Chord（F#5）    ：  F# + C#       = 根音 + 5度        (兩個音)
```

因為**省略了 3 度音**（決定大調/小調的那個音），Power Chord 聽起來**既不大調也不小調** —— 只有「力量」。這就是為什麼它叫「Power」Chord。

### 為什麼用 Power Chord？

1. **搭配 Distortion 效果時聲音乾淨。** 全和弦 + Distortion 會變成一團糊，Power Chord 只有兩個音，破音下依然清晰。
2. **指型固定、移動方便。** 同一個手型在指板上滑來滑去就能變調。
3. **力量感強烈。** 名副其實的「強力」和弦。

---

### 1.1 兩弦 Power Chord（基本型）

```
以 F#5 (6弦根音) 為例：

   6弦  5弦  4弦  3弦  2弦  1弦
    ●    ●    x    x    x    x
    1    3
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠══●═╪════╪════╪════╪════╣  2格   6弦：食指(1) = F# (根音)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  3格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪════╪════╪════╣  4格   5弦：無名指(3) = C# (5度)

按法：
  食指(1) → 6弦第 2 格（F# 音 = 根音）
  無名指(3) → 5弦第 4 格（C# 音 = 5度）
  其餘弦全部不彈（用左手指輕觸消音）
```

### 1.2 三弦 Power Chord（加厚型）

```
以 F#5 (6弦根音三弦版) 為例：

   6弦  5弦  4弦  3弦  2弦  1弦
    ●    ●    ●    x    x    x
    1    3    4
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠══●═╪════╪════╪════╪════╣  2格   6弦：食指(1) = F#
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  3格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪══●═╪════╪════╣  4格   5弦：無名指(3) = C#
                                      4弦：小指(4) = F# (高八度根音)
```

三弦版本多加了一個**高八度的根音**，聲音更飽滿、更厚實。

> [!TIP]
> **本曲建議使用三弦版 Power Chord。** 在 Chorus 段落的全力刷弦中，三弦的飽滿感差異非常明顯。

---

### 1.3 5 弦根音的 Power Chord

有些和弦的根音在 5 弦上，指型完全相同，只是往細弦方向移一組：

```
以 B5 (5弦根音) 為例：

   6弦  5弦  4弦  3弦  2弦  1弦
    x    ●    ●    ●    x    x
         1    3    4
   ╠════╪════╪════╪════╪════╣  1格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪══●═╪════╪════╪════╣  2格   5弦：食指(1) = B (根音)
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪════╪════╪════╣  3格
   ┃    ┃    ┃    ┃    ┃    ┃
   ╠════╪════╪══●═╪══●═╪════╣  4格   4弦：無名指(3) = F#
                                      3弦：小指(4) = B (高八度)
```

> **注意：6弦不彈！** 用食指的指尖輕觸 6 弦消音，或者右手控制只刷 5-4-3 弦。

## 2. Intro Riff（前奏 Riff）

這是「GHOST」的「聲音名片」—— 一聽到這個 Riff 就知道是這首歌。

### Riff 的基本骨架

```
音色：Crunch（微破音）
節拍器：91 BPM (先從 60 BPM 練起)

TAB：（半音下降調弦）

    F#5          B5           C#5          B5
e|------------|------------|------------|------------|
B|------------|------------|------------|------------|
G|------------|------------|------------|------------|
D|--4---------|--9---------|--11--------|--9---------|
A|--4---------|--9---------|--11--------|--9---------|
E|--2---------|--7---------|--9---------|--7---------|
    PM.........  PM.........  PM.........  PM.........

PM = Palm Mute（悶音）
```

> 以上是 Riff 的和弦骨架的簡化版。實際的 Riff 節奏更複雜，在 [06 篇](06_分段教學篇.md) 會詳細拆解每一拍的細節。

### Riff 的節奏骨架

```
拍子：  1    &    2    &    3    &    4    &
右手：  ↓    ↓    ↓    ↓    ↓    ↓    ↓    ↑
悶音：  PM   PM   PM   PM   開   PM   PM   開

PM = 悶音（右手掌輕壓琴橋）
開 = 放開，讓和弦自由響

這個「悶悶悶悶-開-悶悶-開」的模式
就是搖滾 Riff 最經典的律動感來源
```

## 3. Verse 的和弦（Clean Chord Voicing）

Verse 段落使用比較安靜的 Clean 音色。這裡不用 Power Chord，而是用**更豐富的和弦聲位（Voicing）**。

### 常見的 Verse 和弦配置

```
B (Open Shape, Capo 或 Barre):

   Version 1: B Major Barre (2格Barre)
   6弦  5弦  4弦  3弦  2弦  1弦
    x    ●    ●    ●    ●    ●
         1    1    1    1    1    ← 食指橫按 2～7格 (depending on version)

   Version 2: 簡化版 (只彈高音弦)
    x    x    4    4    4    2

   → 在 Clean 段落只要輕撥幾根弦即可
     不需要完整的封閉和弦
```

> [!NOTE]
> **Verse 的彈法非常自由。** 你可以用 Power Chord 加 Clean 音色來簡化，也可以用完整和弦。重點是**輕、薄、安靜** —— 跟 Chorus 形成對比。

## 4. 和弦轉換訓練

### 核心練習 1：Power Chord 直線移動

所有和弦都在 6 弦上，練習準確快速地移動到正確的格位：

```
節拍器：60 BPM → 逐步加到 91 BPM
每個和弦刷 4 拍：

| F#5(2格) | B5(7格) | C#5(9格) | D#5(11格) |
     ↓↓↓↓      ↓↓↓↓      ↓↓↓↓       ↓↓↓↓

⚠️ 重點：移動時手指不要完全離開指板！
        保持食指輕觸弦面滑動，到位後再按下。
```

### 核心練習 2：Intro 和弦快速切換

```
| F#5(2格) | B5(7格) | C#5(9格) | B5(7格) |
     ↓↓↓↓      ↓↓↓↓      ↓↓↓↓      ↓↓↓↓

關鍵轉換：
  F#5 → B5 ：距離 5 格（最大的跳躍）
  B5 → C#5 ：距離 2 格（小移動）
  C#5 → B5 ：距離 2 格（往回滑）
```

### 核心練習 3：帶悶音的和弦移動

```
同上的和弦進行，但加入 Palm Mute：

| F#5(PM) | B5(PM) | C#5(PM) | B5(PM) |

右手掌根始終輕放在琴橋上，移動和弦時手掌不離開。
```

## 5. 消音技巧（Muting）

在電吉他上，**不該響的弦如果發出聲音 = 雜音。** 尤其在 Distortion 音色下，任何碰到弦的振動都會被放大。

### 左手消音

```
按 Power Chord 時：
  食指：按住 6 弦的同時，指尖輕觸 5 弦（如果不需要 5 弦）
        或者用食指的指腹輕觸更高的弦
  不按的弦：左手其他手指輕輕碰觸但不按下 → 消音

目標：只要你不按的弦，全部都是靜音狀態
```

### 右手控制

```
彈 Power Chord 時，Pick 只掃需要的弦：

  6弦根音的 Power Chord → Pick 只掃 6、5、4 弦
  5弦根音的 Power Chord → Pick 只掃 5、4、3 弦

  不要一口氣刷全部 6 根弦！
```
