## 2. 電吉他專屬練習觀念

### 觀念 1：「悶/開」的對比 = 50% 的搖滾感

```
新手的彈法：
  全程開放 → 轟轟轟轟轟轟轟轟  → 沒有層次，像一團噪音

進階的彈法：
  嗒嗒嗒嗒 轟～ 嗒嗒 轟～       → 有壓抑有釋放，像呼吸

Palm Mute 不是裝飾，它是搖滾吉他的「呼吸方式」。
```

### 觀念 2：Distortion 下更需要紀律

```
Clean 音色下：
  碰到不該碰的弦 → 發出小聲雜音 → 勉強可以接受

Distortion 音色下：
  碰到不該碰的弦 → 被放大 10 倍 → 嗡嗡嗡嗡 → 完全不能接受

所以破音越大，你的消音技巧就越重要。
左手的空閒手指要輕觸不彈的弦。
右手要控制 Pick 只掃需要的弦。
```

### 觀念 3：力度比速度重要

```
能 91 BPM 但力度平平 → 聽起來「一般」
能 80 BPM 但力度變化豐富 → 聽起來「很讚」

先把力度做好，再去追速度。
```

### 觀念 4：Solo 可以「自己的版本」

```
原版 Solo 太難？完全可以：
1. 簡化旋律（只彈骨架音）
2. 用 Power Chord 填充 Solo 段落
3. 即興一段自己的旋律

沒有人規定你必須 100% 還原。
音樂的目的是表達，不是考試。
```

## 3. 常見問題 Q&A

### ❓ Q1：我只有木吉他，可以練這首歌嗎？

**可以，但效果有限。** 和弦進行和結構可以在木吉他上學，但：

- Palm Mute 的效果在木吉他上很不一樣
- 沒有 Distortion，Power Chord 聽起來很空
- Solo 段落在木吉他上比較難彈

建議：先在木吉他上學會和弦進行和歌曲結構，等有電吉他後再加入技巧。

---

### ❓ Q2：半音下降調弦會不會傷吉他？

**不會。** 半音下降是降低弦的張力，比標準調弦更溫和。唯一的影響是弦會稍微鬆一點，手感會稍有不同。

如果你經常在標準和半音下降之間切換，弦可能需要重新穩定音準（調完後等幾分鐘再微調一次）。

---

### ❓ Q3：Power Chord 按久了手好痠怎麼辦？

Power Chord 需要的握力比開放和弦大，手部肌肉需要時間適應：

| 原因         | 解法                                   |
| ------------ | -------------------------------------- |
| 握力不足     | 正常，持續練習幾週會改善               |
| 按壓位置不對 | 手指要靠近品格（金屬桿），這樣按力最小 |
| 拇指位置不對 | 拇指放在琴頸後方中央，作為支撐點       |
| 手腕角度太彎 | 確保手腕不要過度彎曲，保持自然角度     |

> **如果是手指痠 → 正常，休息後繼續練。**
> **如果是手腕痛 → 停！檢查姿勢。**

---

### ❓ Q4：Solo 太難了，可以跳過嗎？

**完全可以！** 很多吉他手在 Cover 歌曲時會把 Solo 段落替換成：

1. **繼續刷 Power Chord** → 最簡單的方式
2. **彈 Solo 的旋律骨架**（簡化版）→ 有旋律但不用太高技巧
3. **即興一段自己的東西** → 最有個人風格

不會 Solo 不代表你「不會彈這首歌」。Solo 是加分項。

---

### ❓ Q5：我的 Distortion 聲音很髒、很吵怎麼辦？

常見原因與解法：

| 原因            | 症狀                   | 解法                         |
| --------------- | ---------------------- | ---------------------------- |
| Gain 太高       | 「嘶～～」持續雜音     | 降低 Gain 到 6～7            |
| 消音不好        | 不彈的時候也有聲音     | 左手指輕觸不彈的弦消音       |
| 接地問題        | 摸弦就安靜、放開就嗡嗡 | 檢查導線、可能需要屏蔽拾音器 |
| 沒有 Noise Gate | 停頓時有底噪           | 加一個 Noise Gate 效果       |
| 太靠螢幕 / 燈管 | 電磁干擾               | 離電子設備遠一點             |

---

### ❓ Q6：這首歌適合當第一首電吉他歌曲嗎？

**稍微有點挑戰，但不是不可能。** 難度對比：

| 曲目                        | 難度      | 原因                             |
| --------------------------- | --------- | -------------------------------- |
| Smoke on the Water          | ★★☆☆☆     | 單音 Riff，速度慢                |
| **GHOST**                   | **★★★☆☆** | **Power Chord 移動 + PM + Solo** |
| Through the Fire and Flames | ★★★★★     | 速彈地獄                         |

如果你有耐心用四週慢慢學，GHOST 完全可以當第一首「認真學」的電吉他歌曲。

## 5. 恭喜你！

如果你走到了這裡，你已經：

- ✅ 學會了 Power Chord —— 搖滾吉他的基石
- ✅ 掌握了 Palm Mute —— 讓搖滾「呼吸」的技巧
- ✅ 體驗了 Clean / Crunch / Distortion 三段音色切換
- ✅ 挑戰了吉他 Solo
- ✅ 完整演奏了一首有 11 個段落的搖滾歌曲

這些能力可以直接套用在無數的搖滾歌曲上。

### 下一步建議

1. **學更多搖滾歌曲** — 例如 YOASOBI、ONE OK ROCK、MY FIRST STORY 的曲目，都使用類似的技巧。
2. **進階 Palm Mute** — 學習 Gallop（奔馳節奏）、Djent 等更複雜的悶音技巧。
3. **封閉和弦（Barre Chord）** — 在 Power Chord 的基礎上加入 3 度音。
4. **音階與即興** — 學五聲音階（Pentatonic Scale），開始自己的 Solo 之旅。
