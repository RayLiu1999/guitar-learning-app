## 1. 左手姿勢

### 拇指位置

```
            琴頸截面圖

             ┌─────┐
             │     │  ← 琴頸背面
             │  👍 │  ← 拇指放在這裡（琴頸背面中央偏上）
             │     │
         ────┴─────┴────  ← 指板
         ● ● ● ● ● ●    ← 弦

  ✅ 正確：拇指在琴頸背面中央，作為支撐點
  ❌ 錯誤：拇指勾在琴頸上方（像抓握棒球棒）
           → 這會限制其他手指的伸展範圍
```

> [!NOTE]
> **例外：** 有些搖滾吉他手會用拇指按 6 弦（Jimi Hendrix 風格）。這是進階技巧，初學階段先保持拇指在背面。

### 手指弧度

```
  正確 ✅                        錯誤 ❌

  手指彎曲，用指尖按弦：          手指伸平，用指腹按弦：

     ╭╮                            ──────
     │↓│  ← 指尖垂直壓弦            ↓ 壓弦
  ───●──  ← 弦                  ───●●●──  ← 會碰到鄰弦
     │                              │

  只碰到目標弦                    碰到旁邊的弦 → 雜音
```

### 手指位置

```
按弦時，手指要靠近品格（金屬桿）的右側，而不是正中間：

      品格      品格
       │        │
  ─────┤●───────┤──── ← 弦
       │  ❌     │
       │ 正中間  │
       │按壓力大  │

       │        │
  ─────┤────●───┤──── ← 弦
       │    ✅   │
       │ 靠近品格 │
       │按壓力小  │

靠近品格按壓，需要的力道最小，音最清楚。
```

### 手指編號

```
左手四指的代號（全世界通用）：

  食指 = 1
  中指 = 2
  無名指 = 3
  小指 = 4

  拇指 = T（通常不編號）
```

## 2. 半音階爬格子（Chromatic Exercise）

這是全世界吉他手最常用的暖身練習，沒有之一。

### 基本爬格子（1-2-3-4）

```
節拍器：60 BPM（一拍一個音）
Pick：嚴格交替撥弦 ↓↑↓↑

上行（粗弦→細弦，低把位→高把位方向不動）：

e|-----------------------------------1--2--3--4--|
B|----------------------------1--2--3--4---------|
G|---------------------1--2--3--4----------------|
D|--------------1--2--3--4-----------------------|
A|-------1--2--3--4------------------------------|
E|--1--2--3--4-----------------------------------|
    1  2  3  4
    ↓  ↑  ↓  ↑

指法：
  1格 = 食指(1)    2格 = 中指(2)
  3格 = 無名指(3)  4格 = 小指(4)

每根弦：1-2-3-4
```

```
下行（反向）：

e|--4--3--2--1-----------------------------------|
B|--------------4--3--2--1-----------------------|
G|---------------------4--3--2--1----------------|
D|----------------------------4--3--2--1---------|
A|-----------------------------------4--3--2--1--|
E|----------------------------------------4--3--2--1--|
```

### 重點

1. **每個音都要清晰。** 按不實 = 白練。
2. **手指不要「飛」。** 離開弦的手指保持在弦上方 1cm 以內，不要高高抬起。
3. **一根手指按下去的時候，其他手指不要亂跑。** 這就是「獨立性」。
4. **Pick 嚴格交替。** 這同時在練右手。

## 3. 爬格子的進階變化

### 變化 1：位移爬格子

```
在不同把位重複，每次往上移一格：

第一輪（1-4格）：
E|--1--2--3--4--|

第二輪（2-5格）：
E|--2--3--4--5--|

第三輪（3-6格）：
E|--3--4--5--6--|

一直到 12 格再反向回來。

高把位的間距更小，手指會更擠 → 訓練小指的伸展。
低把位的間距更大 → 訓練手指的跨距。
```

### 變化 2：排列組合

只用 1-2-3-4 四個手指，可以排列出 24 種順序。以下是最常用的幾種：

```
基本：    1-2-3-4    （從食指到小指）
反向：    4-3-2-1    （從小指到食指）
跳指 A：  1-3-2-4    （跳過中指）
跳指 B：  1-4-2-3    （食指→小指跳）
獨立性：  1-3-4-2    （不常用的順序 → 特別訓練獨立性）
```

```
練習方式（以 1-3-2-4 為例）：

E|--1--3--2--4--|
A|--1--3--2--4--|
D|--1--3--2--4--|
G|--1--3--2--4--|
B|--1--3--2--4--|
e|--1--3--2--4--|

格數 = 手指代號（1格=食指，2格=中指，3格=無名指，4格=小指）
```

> [!TIP]
> **哪個排列最難，就多練哪個。** 如果 1-4-2-3 讓你的小指完全不聽話，那就是你的弱點 —— 弱點練好了，進步最大。

### 變化 3：水平爬格子

之前的練習都是「垂直」的（在同一把位，從粗弦爬到細弦）。水平爬格子是沿著**一根弦**從低把位爬到高把位：

```
只用第 6 弦（或任何一根弦）：

E|--1-2-3-4--2-3-4-5--3-4-5-6--4-5-6-7--...--9-10-11-12--|

每 4 個音一組，每組往上移一格。
```

## 4. 手指獨立性專項訓練

### 蜘蛛爬行（Spider Exercise）

```
節拍器：60 BPM

TAB：
e|------------------------------------------1--2--3--4--|
B|-----------------------------------1--2--3--4---------|
G|----------------------------1--2--3--4----------------|
D|---------------------1--2--3--4-----------------------|
A|--------------1--2--3--4------------------------------|
E|--1--2--3--4------------------------------------------|

但每換一根弦時，手指的起始位置不同：

弦   指法
6弦：1-2-3-4  (正常)
5弦：2-3-4-1  (食指最後按)
4弦：3-4-1-2  (無名指先按)
3弦：4-1-2-3  (小指先按！)
2弦：1-2-3-4  (回到正常)
1弦：2-3-4-1  ...
```

### 固定手指練習

```
目的：訓練某根手指獨立運動，其他手指不動

練習 A：固定食指和中指，只動無名指和小指

E|--1--2--3----1--2----3--|  ← 食指中指按住不放
   1  2  放 3   1  2  放 3     3和4號手指在上面活動

（根據你實際的弱點自由組合）
```

## 5. 每日暖身推薦流程

```
以下流程約 5 分鐘，適合每次練琴前執行：

┌─ Step 1（1 分鐘）：空手手指操
│  雙手快速開合 10 次
│  每根手指逐一彎曲伸直
│
├─ Step 2（2 分鐘）：基本爬格子 1-2-3-4
│  上行＋下行各一趟
│  60 BPM，配合交替撥弦
│
├─ Step 3（1 分鐘）：一種排列組合爬格子
│  每天換不同的排列（如今天 1-3-2-4，明天 4-2-3-1）
│
└─ Step 4（1 分鐘）：空弦交替撥弦
   80 BPM，每弦 8 拍，從 6 弦到 1 弦
```
