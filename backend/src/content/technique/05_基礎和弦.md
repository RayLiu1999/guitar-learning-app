## 2. 封閉和弦（Barre Chord）

### 為什麼需要封閉和弦？

```
開放和弦的限制：
  只能彈固定幾個調（E, Em, A, Am, D, C, G 等）
  遇到 F#, Bb, Eb 等調性就無能為力

封閉和弦的優勢：
  一個手型 + 移動位置 = 所有 12 個調全部搞定
  跟 Power Chord 一樣可以在指板上滑動
```

### 封閉和弦 = 食指橫按 + Power Chord 擴展

### 2.1 E 型封閉和弦（6 弦根音）

把 E 和弦 的手型，整個搬到高把位，食指橫按替代「弦枕」：

```
E 和弦（開放）  →  F 和弦（1格橫按）  →  G 和弦（3格橫按）

E:               F:                  G:
0 0 0 0 0 0     ● ● ● ● ● ●       ● ● ● ● ● ●
┠─┼─┤           ├─┼─┤               ├─┼─┤
┠─●─┤           ├─●─┤               ├─●─┤
●─┼─●           ●─┼─●               ●─┼─●
┠─┼─┤           ┠─┼─┤               ┠─┼─┤

                 1格                  3格

E 型手型不變，只是整體往上移，食指橫按所有弦。
```

**指法公式：**

| 手指      | 工作                |
| --------- | ------------------- |
| 食指(1)   | 橫按整格的所有 6 弦 |
| 中指(2)   | 3 弦                |
| 無名指(3) | 5 弦                |
| 小指(4)   | 4 弦                |

### 2.3 大調 vs 小調

相同的概念，把 Em 型搬上去就是小調封閉和弦：

```
Em 型封閉和弦（6 弦根音小調）：

Fm:  1格      Gm:  3格      Am:  5格
● ● ● ● ● ●  以此類推...
├─┼─┤
├─┼─●
●─┼─┤
┠─┼─┤
```

**大調 vs 小調的差別只在一根手指：**

```
E 型大調（如 F）：          E 型小調（如 Fm）：
  中指按 3 弦                 3 弦只有食指橫按（不額外按）
→ 3 弦音比較高               → 3 弦音比較低
→ 大調（明亮）               → 小調（陰暗）
```

## 3. 封閉和弦的練習方法

### 食指橫按的要訣

封閉和弦最大的挑戰就是**食指橫按**。以下是讓它成功的秘訣：

```
1. 用食指的側面（不是正面）：
   ───────────          ───────────
   即使正面的肉有「溝」  側面的骨頭比較硬
   會讓弦卡在溝裡       弦壓得更穩
   ❌                    ✅

2. 食指稍微靠近品格：
   ───┤●────┤──          ───┤────●┤──
       ❌ 正中間               ✅ 靠品格

3. 手肘往身體方向收：
   這會讓食指自然旋轉到側面，增加壓力。

4. 拇指在琴頸背面正中央施力：
   拇指和食指「對捏」—— 拇指的力量是食指橫按的「後盾」。
```

### 食指橫按練習

```
不按其他手指，只練食指橫按：

1. 食指橫按 1 格的所有弦
2. 右手從 6 弦到 1 弦逐一撥弦
3. 每根弦都要有清晰的音（不是悶的「嗒」聲）
4. 如果某根弦不響 → 微調食指位置 / 增加壓力 / 檢查側面角度
5. 1 格成功後，移到 5 格（間距更小，更容易）
```

### 完整封閉和弦練習

```
節拍器：40 BPM（一小節按一個和弦）

| F(1格) | G(3格) | A(5格) | G(3格) |

每個和弦：
  1. 按下去
  2. 逐弦檢查清晰度
  3. 刷一次確認整體聲音
  4. 鬆開
  5. 移位
  6. 重複

速度不重要，清晰度才重要。
```

## 4. Power Chord → Barre Chord 升級路徑

```
階段 1（初級）：Power Chord
  A5: 食指 + 無名指（或小指），只按 2～3 根弦
  → 用在 Distortion 重 Riff

階段 2（中級）：Barre Chord
  A: 食指橫按 + 中指/無名指/小指，按 5～6 根弦
  → 用在 Clean 和弦、Funk 節奏、歌曲伴奏

階段 3（進階）：自由切換
  同一首歌的同一段落，根據音色需求
  Distortion = Power Chord
  Clean = Barre Chord
  → 這就是真正的節奏吉他手
```

## 5. 指板上的和弦系統

### 同一個和弦有多種按法

```
A 和弦至少有 3 種常見位置：

  A (開放)         A (E型, 5格)       A (A型, 12格)
  x 0 ● ● ● 0     ● ● ● ● ● ●       x ● ● ● ● ●
    ┠─┼─┤           ├─┼─┤               ├─┼─┤
    ┠─●─●           ├─●─┤               ├─●─●
    ┠─●─┤           ●─┼─●               ┠─●─┤
    ┠─┼─┤           ┠─┼─┤               ┠─┼─┤

  0格              5格                 12格
```

### CAGED 系統（預告）

電吉他的和弦系統有一個叫做 **CAGED** 的記憶法，它用 C、A、G、E、D 五種開放和弦的「型」來覆蓋整個指板。這是進階的和弦理論，等你熟悉封閉和弦後可以深入研究。
