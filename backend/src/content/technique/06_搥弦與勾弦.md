# 06 — 搥弦與勾弦（Hammer-on & Pull-off / Legato）

> 不靠 Pick，只靠左手就能讓弦發聲。這是通往 Solo 的第一道門。

---

## 前置需求

- [02 左手基礎](02_左手基礎與爬格子.md)

## 你會學到

- ✅ Hammer-on（搥弦）— 手指「敲下去」發聲
- ✅ Pull-off（勾弦）— 手指「勾離開」發聲
- ✅ 連續 H-P（Legato Run）
- ✅ Trill（顫音）

---

## 1. Hammer-on（搥弦）

### 原理

```
Step 1: 右手先撥一個音（如 1 弦 5 格）
Step 2: 左手另一根手指**用力敲擊**更高的格位（如 1 弦 7 格）
Step 3: 不需要右手再撥，弦就會因為敲擊而振動 → 第二個音響了

記法：5h7  =  5 格 Hammer-on 到 7 格
```

### 關鍵

- **敲擊要快、準、有力。** 像彈鋼琴一樣「啪」地按下去。
- **手指要落在品格旁邊（右側）。** 跟普通按弦的位置一樣。
- **如果第二個音太小聲** → 你的敲擊力度不夠，或位置不對。

### 練習

```
節拍器：60 BPM

1弦練習：
e|--5h7----5h7----5h7----5h7----|
  ↓       ↓       ↓       ↓
  只有第一個音撥弦，第二個音靠搥弦

逐漸嘗試不同手指組合：
  1→2 (食指→中指)：5h6
  1→3 (食指→無名指)：5h7
  1→4 (食指→小指)：5h8   ← 最難，小指力量最弱

以及不同弦：6弦的搥弦聲音最大（弦最粗），1弦最小。
```

## 2. Pull-off（勾弦）

### 原理

```
Step 1: 左手同時按住兩個格位（如食指按 5 格，無名指按 7 格）
Step 2: 無名指**向下勾離**弦面（不是直接抬起！）
Step 3: 弦被勾動後振動 → 食指的 5 格音響了

記法：7p5  =  7 格 Pull-off 到 5 格
```

### 關鍵

```
  ❌ 直接抬起：          ✅ 向下勾：
     ↑ 手指往上抬            ↙ 手指向下（或向掌心）勾
     弦不振動                弦被勾動，振動發聲

Pull-off 不是「放開」，是「撥弦」—— 用左手手指撥弦。
```

### 練習

```
節拍器：60 BPM

1弦練習：
e|--7p5----7p5----7p5----7p5----|
  ↓       (不撥)  (不撥)  (不撥)
  第一下撥弦，之後只靠 Pull-off

手指組合：
  3→1 (無名指→食指)：7p5
  2→1 (中指→食指)：6p5
  4→1 (小指→食指)：8p5
```

## 3. H-P 連音（Legato Run）

把 Hammer-on 和 Pull-off 串起來，就是連續的圓滑音：

```
節拍器：60 BPM

e|--5h7p5----5h7p5----5h7p5----5h7p5----|
   ↓         (不撥)    (不撥)    (不撥)

只有第一個音用 Pick，後面全靠左手。
每組 5h7p5 是三個音。

進階（四音一組）：
e|--5h6h7p6----5h6h7p6----5h6h7p6----|
   ↓

五音一組：
e|--5h6h7h8p7p6p5----|
   ↓
```

### Legato vs Picking 的音色差異

```
Picking（每個音都用 Pick 撥）：
  音色銳利、有攻擊感、每個音的起始「啪」聲很明顯
  聽起來像：Ta-Ta-Ta-Ta

Legato（用 H-P 連接）：
  音色圓滑、流暢、音與音之間像「黏在一起」
  聽起來像：Da-a-a-a

兩種都要會 → 才能選擇用哪種風格。
```

## 4. Trill（顫音）

快速重複 Hammer-on 和 Pull-off：

```
Trill 記法：5tr7  或  5~~~~

e|--5h7p5h7p5h7p5h7p5h7p5h7----|

   快速重複 H-P，像弦在「顫抖」

速度從慢到快，像引擎啟動：
   5h7p5.....h7p5...h7p5..h7p5.h7p5h7p5h7p5h7p5
   慢 ─────────────────────────────────── 快
```

### Trill 練習

```
每對手指各 Trill 15 秒：

  1-2 (食中)：5tr6    → 15 秒
  1-3 (食無名)：5tr7  → 15 秒
  1-4 (食小)：5tr8    → 15 秒  ← 最累！
  2-3 (中無名)：6tr7  → 15 秒
  2-4 (中小)：6tr8    → 15 秒
  3-4 (無名小)：7tr8  → 15 秒  ← 獨立性大考驗

哪對手指的 Trill 最不穩定，就是你的弱點。
```
