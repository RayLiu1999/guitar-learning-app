## 1. 推弦（Bending）

### 原理

按住弦，手指把弦**往上推**（或往下拉），讓弦的有效長度縮短 → 音高升高。

```
推弦前：                    推弦後：
弦 ───────●────────         弦 ────────╲
          ↑                            ╲●  ← 弦被推高
          正常位置                       ↑
                                     推高的位置

半音推弦：音升高半音（如 7 格的音升到 8 格的音高）
全音推弦：音升高全音（如 7 格的音升到 9 格的音高）
```

### 推弦的正確用力方式

```
❌ 只用一根手指用力推 → 手指很快就累壞

✅ 正確：多指輔助 + 手腕旋轉

  無名指推弦時的手指分配：
   ╭─ 食指：放在 5 格，輔助推力
   ├─ 中指：放在 6 格，輔助推力
   └─ 無名指：放在 7 格，主力推弦   ← 三根手指一起推！

  力量來源 = 手腕旋轉（像轉門把），不是手指掙扎
```

### 練習 A：半音推弦

```
1 弦，7 格（無名指按）

Step 1: 先撥 8 格的音，記住它的音高
Step 2: 回到 7 格，推弦，推到跟 8 格一樣的音高
Step 3: 確認音準 → 如果太低就多推一點，太高就少推一點

e|--8----7b8----8----7b8----|
   ↑     ↑      ↑     ↑
   參考   推弦   參考   推弦

節拍器：自由速度（先不管節拍，專注音準）
```

### 練習 B：全音推弦

```
1 弦，7 格

Step 1: 先撥 9 格的音，記住音高
Step 2: 回到 7 格，推弦推到 9 格的音高（比半音推更遠）

e|--9----7b9----9----7b9----|

全音推弦需要的力度大約是半音推弦的兩倍。
3 弦（G 弦）的全音推弦是最常用的。
```

### 常見問題

| 問題             | 原因         | 解法                             |
| ---------------- | ------------ | -------------------------------- |
| 音推不準         | 沒有參考音   | 每次推弦前先聽目標音             |
| 推到一半沒力     | 只用一根手指 | 三指輔助 + 手腕旋轉              |
| 弦滑出指尖       | 推弦角度不對 | 手指要「卡住」弦，不是「滑過」弦 |
| 只有 1、2 弦能推 | 6 弦太粗     | 正常，粗弦的推弦確實更難         |

## 2. Vibrato（揉弦）⭐

### 為什麼 Vibrato 這麼重要？

```
同一個音，沒有 Vibrato：
  「嗯─────────────────」 ← 平平的，像機器在響

同一個音，加了 Vibrato：
  「嗯～～～～～～～～～～」 ← 有生命力，像人在唱歌

Vibrato 就是在一個音上做微小的、有節奏的推弦。
音高在「原音」和「稍高」之間輕微搖擺。
```

> [!IMPORTANT]
> **Vibrato 是區分業餘和職業吉他手最明顯的標誌。** 職業吉他手的 Vibrato 速度均勻、幅度一致、聽起來像在「唱歌」。業餘的 Vibrato 要嘛太快像在抖、要嘛不均勻像在抽搐。

### 兩種 Vibrato

```
手腕式 Vibrato（推薦）：

  力量來源 = 手腕旋轉
  手指幾乎不動，是手腕帶動整個手在旋轉
  像轉門把：順時針（推）→ 逆時針（回）→ 順時針 → ...

  ✅ 穩定、可控、速度和幅度容易調整


手指式 Vibrato（古典吉他風格）：

  力量來源 = 手指左右搖晃
  手腕不動，手指在弦上「揉」
  像揉泥土

  ⚠️ 在電吉他上效果較弱，但某些情況也會用到
```

### Vibrato 練習

```
練習 A：慢速 Vibrato

1 弦 7 格，無名指按住：

節拍器：60 BPM
每個八分音符做一次「推→回」：

拍子：  1    &    2    &    3    &    4    &
音高：  ↑    ↓    ↑    ↓    ↑    ↓    ↑    ↓
        推   回   推   回   推   回   推   回

推的幅度 ≈ 四分之一音（很小），要均勻。


練習 B：加速 Vibrato

同上，但加快到每個十六分音符一次：

拍子：  1  e  &  a    2  e  &  a
音高：  ↑  ↓  ↑  ↓    ↑  ↓  ↑  ↓

⚠️ 加速時，幅度會自然變小，這是正常的。
   關鍵是保持均勻。


練習 C：長音 Vibrato

彈一個音，持續 4 秒的 Vibrato：

e|--7～～～～～～～～～～～～～～|

嘗試在 4 秒內保持速度和幅度一致。
用手機錄下來聽 → 是否均勻？
```

## 3. 進階推弦技巧

### Release Bend（推弦回放）

```
先推到目標音高，然後慢慢放回來：

e|--7b9r7--|
    ↑  ↑
    推  回  ← 放回原音

聽起來像「嗚～噢～」
```

### Pre-Bend（預推弦）

```
先把弦推到目標音高（不發聲），然後撥弦，再放回來：

e|--(7b9)r7--|
    ↑     ↑
    先推   再放

聽起來像「噢～」（從高音開始往下滑）
```

### Unison Bend（齊音推弦）

```
一根弦按住不動（目標音），另一根弦推到相同的音高：

e|------7------|
B|--7b9--------|

2 弦 7 格推到 9 格的音高 = 1 弦 7 格的音
兩個音同時響起，音高相同 → 「嗡～」的共振效果
```

## ✅ 本篇檢查清單

- [ ] 半音推弦音準正確（推到的音高跟比較高一格的音一致）
- [ ] 全音推弦音準正確（推到的音高跟比較高兩格的音一致）
- [ ] 推弦時使用三指輔助 + 手腕旋轉（不是單指硬推）
- [ ] Vibrato 能持續 4 秒且速度均勻
- [ ] 能做出 Release Bend（推→回）
