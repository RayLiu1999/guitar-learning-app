# 08 — 滑音（Slide）

> 音符之間的「潤滑劑」。讓旋律不只是跳，而是流動。

## 1. 基本滑音

### 上行滑音（Slide Up）

```
記法：5/7  或  5s7

Step 1: 左手按住某格（如 5 格），右手撥弦
Step 2: 手指不離開弦面，沿著弦滑到目標格（7 格）
Step 3: 滑動的過程中弦持續振動 → 聽到音高連續上升

e|--5/7----|

手指保持按壓力，在弦上「滑行」。
如果手指離開弦面 → 聲音會中斷。
如果手指壓力不夠 → 聲音會變小或消失。
```

### 下行滑音（Slide Down）

```
記法：7\5  或  7s5

e|--7\5----|

同上，只是方向反過來。
音高從高往低連續下降。
```

### 練習

```
節拍器：60 BPM

練習 A：固定距離滑音
e|--5/7----7\5----5/7----7\5----|
    ↓       (不撥)  ↓      (不撥)

每兩拍做一次滑音（第一拍滑到，第二拍停留）。

練習 B：不同距離
e|--3/5----3/7----3/9----3/12----|
   2格距    4格距   6格距    9格距

距離越遠，滑動速度越快，需要更好的控制。

練習 C：不同弦
在 6 弦到 1 弦每根弦上各做 5/7 和 7/5 的滑音。
粗弦（6弦）滑音阻力大，細弦（1弦）阻力小，手感不同。
```

## 2. 滑入與滑出

### 滑入（Slide In / Grace Note Slide）

```
從不確定的位置滑到目標音：

記法：/7（從下方滑入）  或  \7（從上方滑入）

e|--/7----|     e|--\7----|
   從低處滑到7     從高處滑到7

起始位置不固定，看手感。
效果是一種「滑進來」的裝飾性起手。
```

### 滑出（Slide Out）

```
從某個音滑到不確定的位置然後消失：

記法：7/（往上滑出）  或  7\（往下滑出）

e|--7/----|     e|--7\----|
   7格往上滑消失    7格往下滑消失

在結尾使用，像聲音慢慢「溜走」。
```

## 3. 長距離滑音（把位跳躍）

```
e|--2/12----|     （從 2 格滑到 12 格，距離 10 格！）

長距離滑音的技巧：
1. 手指壓力要足（不能太輕，否則聲音到一半就沒了）
2. 速度要連貫（不要中途減速）
3. 到達目標格時精準停住

用途：
  - 段落過渡時的「滑行」效果
  - 在 Riff 中製造戲劇性
  - 從低把位轉到高把位（例如進入 Solo）
```

## 4. Slide vs Bend：什麼時候用哪個？

| 比較       | Slide                              | Bend                       |
| ---------- | ---------------------------------- | -------------------------- |
| 音高變化   | 經過所有中間音（像唱歌的「滑音」） | 只有起點和終點（直接拉到） |
| 適合的距離 | 大距離（2 格以上）                 | 小距離（半音或全音）       |
| 音色       | 流暢、連續                         | 情感、張力                 |
| 手指位置   | 會改變（滑到新位置）               | 不變（同一格推弦）         |
| 搭配       | 常搭配 Legato（H-P）               | 常搭配 Vibrato             |

```
例：從 A 音到 B 音（差一個全音）

  Slide 版：  e|--5/7----| ← 聽到 5→6→7 的連續音
  Bend 版：   e|--5b7----| ← 直接從 5 拉到 7 的音高

  哪個好？看場景：
    旋律性段落 → Slide
    情感爆發段落 → Bend
```

---

## 5. Slide 在實際歌曲中的應用

```
範例 1：Riff 中的裝飾
e|--/5--5--7--5--|   ← 第一個音用滑入

範例 2：Solo 中的連接
e|--7/9--9--12\9--|  ← 滑上去再滑回來

範例 3：段落過渡
e|--12\2----------|  ← 長距離滑下來，進入下一段

範例 4：Power Chord 滑動
6弦：|--2/7------| ← 整個 Power Chord 一起滑（F#5 滑到 B5）
5弦：|--4/9------|
```
