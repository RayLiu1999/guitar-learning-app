# 09 — 音階系統

> 音階是音樂的字母表。學了字母才能組成單詞，寫出句子，最終即興演說。

## 1. 為什麼從五聲音階開始？

```
全音階有 7 個音  → 選擇太多，容易彈出「難聽」的音
五聲音階只有 5 個音 → 怎麼組合都好聽！

幾乎所有搖滾、藍調、流行 Solo 都建立在五聲音階上。
如果只學一個音階 → 選這個。
```

## 2. A 小調五聲音階（Am Pentatonic）

### 組成音

```
A - C - D - E - G

只有 5 個音，沒有半音的「摩擦感」，
所以怎麼連怎麼跳都很和諧。
```

### 把位 1（最重要！）

```
5 格位置：

e|--5-----8----|
B|--5-----8----|
G|--5--7-------|
D|--5--7-------|
A|--5--7-------|
E|--5-----8----|

   ●           ●        ← 食指(1)按 5 格
         ●               ← 無名指(3)按 7 格
               ●         ← 小指(4)按 8 格

指法圖（○ = 音符位置）：

   5格  6格  7格  8格
e|--○-----------○--|
B|--○-----------○--|
G|--○-----○--------|
D|--○-----○--------|
A|--○-----○--------|
E|--○-----------○--|

把位 1 是最常用的，你會在 80% 的搖滾 Solo 中看到它。
```

### 練習方法

```
節拍器：60 BPM
交替撥弦 ↓↑

上行（低音→高音）：
E|--5--8----|
A|--5--7----|
D|--5--7----|
G|--5--7----|
B|--5--8----|
e|--5--8----|

下行（高音→低音）：
e|--8--5----|
B|--8--5----|
G|--7--5----|
D|--7--5----|
A|--7--5----|
E|--8--5----|

上行下行算一次，連續跑 4 次不中斷。
```

### 把位 2～5（簡圖）

```
把位 2 (7-8格)     把位 3 (9-10格)    把位 4 (12格)      把位 5 (2-3格)

e|--8--10--|        e|--10-12--|       e|--12-13--|       e|--3--5--|
B|--8--10--|        B|--10-13--|       B|--12-13--|       B|--3--5--|
G|--7--9---|        G|--9--12--|       G|--12-14--|       G|--2--5--|
D|--7--10--|        D|--10-12--|       D|--12-14--|       D|--2--5--|
A|--7--10--|        A|--10-12--|       A|--12-14--|       A|--3--5--|
E|--8--10--|        E|--10-12--|       E|--12-15--|       E|--3--5--|
```

> [!TIP]
> **先只學把位 1，把它練到閉眼都能跑。** 把位 2～5 等你把位 1 很熟之後再學。一次學太多把位反而什麼都記不住。

## 3. 五聲音階的「移調」

小調五聲音階的形狀是固定的，只要移動位置就能換調：

```
Am Pentatonic：把位 1 在 5 格（A 音在 6 弦 5 格）
Em Pentatonic：把位 1 在 0 格（開放弦）/ 12 格
Bm Pentatonic：把位 1 在 7 格
F#m Pentatonic：把位 1 在 2 格  ← GHOST 那首歌！
```

---

## 4. 音階的「跑法」

光上下行太無聊。以下是幾種常見的音階跑法，讓音階變成「音樂」：

### 跑法 1：三度跑

```
不是 1-2-3-4-5，而是 1-3, 2-4, 3-5, 4-6...（每次跳一個音）

TAB（Am Penta 把位 1，6弦+5弦部分）：
E|--5--8----|
A|--5-------|
E|--8-------|
A|--5--7----|
E|----------| ...

音符：A-C, C-D, D-E, E-G, ...
```

### 跑法 2：四音一組

```
1-2-3-4, 2-3-4-5, 3-4-5-6...

TAB：
E|--5-8----|
A|--5------|
E|--8------|
A|--5-7----|
D|--5------|
A|--7------| ...
```

### 跑法 3：下行小樂句

```
從高音開始，每次下降後退一個音再上行：

e|--8-5----|
B|------8--|
e|--5------|
B|--8-5----|
G|------7--|
B|--5------| ...

這種「鋸齒狀」的跑法很常出現在搖滾 Solo 中。
```

## 5. 大調 / 小調音階（簡介）

### 自然大調音階

```
C 大調：C - D - E - F - G - A - B（7 個音）

全-全-半-全-全-全-半  （音程關係）

比五聲音階多了 2 個音（F 和 B），
聽起來更「完整」但也更容易彈出不和諧的音。
```

### 自然小調音階

```
A 小調：A - B - C - D - E - F - G（7 個音）

全-半-全-全-半-全-全  （音程關係）

跟 C 大調用的音完全相同！（只是起始音不同）
這叫做「關係大小調」。
```

### 五聲 vs 七聲音階

```
Am Pentatonic：A - C - D - E - G      （5 個音）
Am Natural：   A - B - C - D - E - F - G  （7 個音）
                   ↑               ↑
                   多了 B 和 F

B 和 F 是「危險音符」（放在錯誤的和弦上會很不和諧）。
但如果用對了，它們會增加旋律的色彩和張力。
```

> 建議路線：先精通五聲 → 再逐漸加入第 6、第 7 個音 → 自然過渡到七聲。
