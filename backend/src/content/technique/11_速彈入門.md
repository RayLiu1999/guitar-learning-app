## 1. 速度的三大瓶頸

```
瓶頸 1：手腕太緊（最常見）
  症狀：速度到某個 BPM 就上不去，手開始抖
  原因：動作幅度太大、肌肉緊繃
  解法：刻意縮小幅度（Pick 穿過弦 ≤ 2mm）、放鬆手腕

瓶頸 2：左右手不同步
  症狀：速度一快，左手按了但右手沒撥（或反過來）
  原因：兩手的「時間差」在高速下被放大
  解法：慢速練到完美同步，再逐步加速

瓶頸 3：跨弦猶豫
  症狀：在同一根弦上可以很快，跨弦時一慢下來
  原因：Pick 在兩根弦之間移動需要額外的動作
  解法：專門練跨弦的交替撥弦
```

## 2. 節拍器漸進訓練法

### 無腦但極其有效的練法

```
Step 1: 找到你的「舒適速度」（能完美⽤交替撥弦跑音階的最快 BPM）
Step 2: 把節拍器設在舒適速度
Step 3: 完美彈完 4 遍 → 加 2 BPM
Step 4: 完美彈完 4 遍 → 再加 2 BPM
Step 5: 到了「開始出錯」的速度 → 退回 4 BPM → 那就是今天的上限
Step 6: 明天重來，從前天的舒適速度開始

⚡ 每天能進步 2～4 BPM 就是巨大的進步。
   一個月 = 60～120 BPM 的提升。
```

### 速度建設日誌（範例）

```
Day 1:  舒適 80 BPM → 加到 86 → 在 88 出錯 → 今日上限 84
Day 2:  從 80 → 加到 88 → 90 出錯 → 今日上限 86
Day 3:  從 82 → 加到 90 → 穩定！→ 今日上限 90
...
Day 14: 從 100 → 加到 110 → 今日上限 108
```

> [!TIP]
> **記錄你每天的上限 BPM。** 看到數字在成長，動力就來了。

## 3. 速彈專項練習

### 練習 A：單弦速度

```
1 弦上的 Am Pentatonic 片段：
e|--5-8-5-8-5-8-5-8--|

只在一根弦上重複兩個音。
消除跨弦的變數，純練 Pick 速度。
```

### 練習 B：跨弦速度

```
1-2 弦跨弦：
e|--5--8-----|
B|------8--5-|
e|--5--8-----|
B|------8--5-|

每跨一次弦，Pick 都要精準移動。
```

### 練習 C：音階速度

```
Am Pentatonic 把位 1 完整上下行：
（從 60 BPM 開始，用十六分音符）

逐漸加速直到你的極限。
```

## 4. Economy Picking 簡介

```
Alternate Picking：永遠 ↓↑↓↑，即使跨弦
Economy Picking：跨弦時，用相同方向的 Pick 掃過去

例：從 2 弦往 1 弦移動（往細弦方向）

  Alternate：  B| ↓ ↑ |  e| ↓ ↑ |  ← 2弦結尾↑，1弦開頭↓
  Economy ：   B| ↓ ↑ |  e| ↑ ↑ |  ← 2弦結尾↑，1弦開頭也↑（順勢帶過去）

Economy Picking 在順向跨弦時更省力。
但打破了 Alternate 的均勻節奏感。

建議：先精通 Alternate Picking 再學 Economy。
```
