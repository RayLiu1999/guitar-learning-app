## 1. 基本 Tapping

```
Step 1: 右手的中指（或食指）在指板上的高把位「敲」下去（= 右手的 Hammer-on）
Step 2: 右手手指「勾離」弦面（= 右手的 Pull-off）→ 左手按的音響了
Step 3: 左手繼續做 H-P

記法：T = 右手 Tap

e|--T12-p8-h5--|
    右手  左手
    敲12  勾到8  搥到5
```

### 練習

```
1 弦，左手食指按 5 格，無名指按 8 格：

e|--T12-p8-p5--T12-p8-p5--T12-p8-p5--|
    ↑     ↑  ↑
    右手   左手Pull 到左手Pull

節拍器：60 BPM（三連音，每拍 3 個音）

關鍵：
  - 右手的 Tap 要用力，音量要跟左手的 H-P 差不多
  - 右手的 Pull-off 要往下勾（不是直接抬起）
  - 不用的時候 Pick 夾在其他手指之間
```

---

## 2. Tapping 琶音

```
在不同弦上 Tapping，形成跨弦的琶音：

Am 琶音 Tapping：
e|--T12-p8-p5---------------------------------|
B|--------------T13-p8-p5---------------------|
G|--------------------------T12-p9-p5---------|

每根弦做一組 T-p-p，然後移到下一根弦。
```

---

## 3. 常見問題

| 問題                 | 解法                           |
| -------------------- | ------------------------------ |
| 右手 Tap 聲音太小    | 敲擊更用力、指尖對準品格       |
| 右手 Pull-off 沒聲音 | 要「勾」不是「抬」             |
| 多餘的弦有雜音       | 左手空閒手指輕觸消音           |
| Pick 不知道放哪      | 夾在拇指和食指之間，用中指 Tap |
