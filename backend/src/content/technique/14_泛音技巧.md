## 1. 自然泛音（Natural Harmonics）

### 原理

弦在特定的分割點上，輕觸（不按下）再撥弦，會產生高頻的「鈴聲」。

```
最容易出泛音的位置：

  弦枕                                           琴橋
  │                                               │
  ├───5格───┤───7格───┤───12格───┤──────────
              ↑           ↑            ↑
              五度泛音     四度泛音     八度泛音
              （中等亮）   （很亮）     （最容易）

12 格 → 弦的正中間 → 最容易、最亮的泛音
7 格  → 弦的 1/3 處 → 稍難但常用
5 格  → 弦的 1/4 處 → 更難，音更高
```

### 做法

```
Step 1: 左手手指輕觸（不按下！）弦的正上方，剛好在品格金屬桿的上方
Step 2: 右手撥弦
Step 3: 撥弦後立刻把左手手指離開弦面
Step 4: 聽到「叮～」的鈴聲 = 成功！

關鍵：
  ✅ 輕觸 → 不是按下去
  ✅ 位置精準 → 在品格金屬桿的正上方（不是格子中間）
  ✅ 撥弦後立刻放開
```

### 練習

```
12 格泛音（最簡單）：

在每根弦的 12 格做自然泛音：
E|--<12>--|
B|--<12>--|
G|--<12>--|
D|--<12>--|
A|--<12>--|
e|--<12>--|

< > = 泛音記號

然後嘗試 7 格和 5 格。
```

## 2. 人工泛音（Pinch Harmonics）⭐

### 這是什麼聲音？

搖滾和金屬中常聽到的「尖叫音」/ 「吱～」聲。像是吉他在叫喊。

### 做法

```
Step 1: 正常彈一個音
Step 2: 在 Pick 撥弦的瞬間，拇指的皮膚碰觸弦
       （Pick 穿過弦時，拇指的肉「蹭」到弦）
Step 3: 弦因為被拇指碰觸，在特定位置被「分割」→ 產生泛音

       ╭── 拇指邊緣
       │
  ───╲│╱─── 弦
       Pick

關鍵：
  Pick 露出要少（1～2mm），這樣拇指自然會碰到弦
  搭配 Distortion 效果更明顯
  不同的撥弦位置會產生不同音高的泛音
```

### 練習

```
Step 1: 調到 Distortion 音色（Gain 6～7）
Step 2: 按住 3 弦 7 格
Step 3: 用很少的 Pick 露出量撥弦
Step 4: 嘗試在不同位置（靠琴橋 / 靠琴頸 / 中間）撥弦
Step 5: 聽到「吱～」就是成功了

如果出不來 → 調整 Pick 露出量（更少） / 調整拇指碰弦的角度
```
