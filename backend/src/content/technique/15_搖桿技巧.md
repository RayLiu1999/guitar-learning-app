# 15 — 搖桿技巧（Whammy Bar / Tremolo Arm）

> 讓音高像液體一樣流動。搖桿把吉他變成「會叫的動物」。

---

## 前置需求

- [07 推弦與揉弦](07_推弦與揉弦.md)
- 你的吉他必須有搖桿系統（Tremolo Bridge / Floyd Rose）

## 你會學到

- ✅ 搖桿的基本操作
- ✅ Dive Bomb（俯衝炸彈）
- ✅ 搖桿 Vibrato

---

## 1. 搖桿系統的類型

| 類型            | 代表吉他         | 能做什麼            | 穩定度 |
| --------------- | ---------------- | ------------------- | ------ |
| Vintage Tremolo | Fender Strat     | 只能壓（降音）      | 中等   |
| Floyd Rose      | Ibanez / Jackson | 可壓可拉（降/升音） | 很高   |
| Bigsby          | Gretsch / Gibson | 微壓（輕微降音）    | 高     |

---

## 2. Dive Bomb（俯衝炸彈）

```
彈一個音 → 用力壓下搖桿 → 音高急遽下降 → 直到弦完全鬆弛

e|--12～～～～↘↘↘↘↘↘↘↘↘↘|
              └── 搖桿壓下 ──→ 音高一路下降到最低

聽起來像炸彈俯衝：「嗚～～～～嗡」

搭配 Distortion + Delay 效果最佳。
```

---

## 3. 搖桿 Vibrato

```
比手指 Vibrato 更寬、更戲劇化的振動效果。

彈一個音 → 輕輕上下搖動搖桿 → 音高在原音附近搖擺

e|--12～～～～～～|
      ↕↕↕↕↕↕  ← 搖桿微幅搖動

適合長音的收尾，增加「歌唱感」。
```

---

## 4. Flutter（顫抖）

```
彈一個音 → 輕敲搖桿讓它自由振動 → 搖桿自己「顫抖」→ 音高快速微幅搖擺

聽起來像「嗡嗡嗡嗡嗡～」機械般的顫動。
需要 Floating Bridge（浮動琴橋）才能做到。
```

---

## 5. 注意事項

```
⚠️ 調音穩定性：
   頻繁使用搖桿會導致弦走音。
   Floyd Rose 有鎖定弦枕，穩定性最好。
   Vintage Tremolo 走音較多，需要常調音。

⚠️ 弦的壽命：
   大幅度的 Dive Bomb 會加速弦的疲勞。
   如果弦開始走音嚴重或音色變悶 → 該換弦了。

⚠️ 不是所有吉他都有搖桿：
   如果你的吉他沒有 Tremolo Bridge → 跳過本篇，完全不影響學習。
```
