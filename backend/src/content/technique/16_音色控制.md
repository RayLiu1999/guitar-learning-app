# 16 — 音色控制

> 同一把吉他、同一顆音箱，只靠雙手就能切換出 5 種以上的音色。

---

## 前置需求

無。買了電吉他之後隨時可以讀。

## 你會學到

- ✅ 吉他上 Volume / Tone 旋鈕的活用
- ✅ 拾音器切換
- ✅ Pick 攻擊角度對音色的影響
- ✅ 手指觸弦位置的影響

---

## 1. Volume 旋鈕

```
不只是調大小聲！Volume 旋鈕還能控制破音程度。

音箱 Gain 設在 6（中高破音）：

  吉他 Volume = 10  → 訊號最大 → 音箱完全破音 → Distortion
  吉他 Volume = 7   → 訊號減弱 → 破音程度降低 → Crunch
  吉他 Volume = 3   → 訊號很弱 → 幾乎不破      → Clean

→ 不用踩踏板，轉 Volume 旋鈕就能在 Clean/Crunch/Dist 之間切換！
→ 很多職業吉他手（如 Angus Young）就是用這招。
```

---

## 2. Tone 旋鈕

```
Tone = 高頻（Treble）的控制

  Tone = 10  → 高頻全開 → 聲音明亮、銳利、有攻擊性
  Tone = 5   → 高頻減半 → 聲音溫暖、柔和
  Tone = 0   → 高頻全關 → 聲音悶悶的、像在毛毯裡彈

使用建議：
  Solo → Tone 7～10（穿透力強）
  Clean 和弦 → Tone 5～7（溫暖）
  Jazz 風格 → Tone 0～3（圓潤）
```

---

## 3. 拾音器（Pickup）切換

```
大多數電吉他有 2～3 組拾音器和一個切換開關：

  ┌──────────┐      ┌──────────┐
  │  Bridge   │      │   Neck   │   ← 兩組拾音器
  │ (琴橋)    │      │  (琴頸)  │
  └──────────┘      └──────────┘
       ↑                  ↑
     明亮、銳利          溫暖、圓潤
     Riff、Power Chord   Clean、Solo

5 段切換開關（常見於 Strat 型吉他）：
  1 = 琴橋拾音器（最亮）
  2 = 琴橋 + 中間混合
  3 = 中間拾音器
  4 = 中間 + 琴頸混合
  5 = 琴頸拾音器（最暖）
```

---

## 4. Pick 的攻擊角度

```
Pick 平行於弦：     Pick 傾斜 45°：
  聲音明亮、銳利       聲音柔和、圓潤
  攻擊感強             滑順
  適合 Riff            適合 Solo 旋律
```

---

## 5. 右手觸弦位置

```
靠琴橋撥弦：           靠琴頸撥弦：
  聲音明亮、緊繃         聲音溫暖、飽滿
  有「叮」的金屬感       有「嗡」的共鳴感
  適合 Riff、PM          適合 Clean、Solo
```

---

## 音色配方表

| 想要的音色  | Volume | Tone | 拾音器      | Pick 位置 |
| ----------- | ------ | ---- | ----------- | --------- |
| 明亮 Riff   | 10     | 8~10 | Bridge      | 靠琴橋    |
| 溫暖 Clean  | 3~5    | 5~7  | Neck        | 中間      |
| 搖滾 Crunch | 7      | 7    | Bridge      | 中間      |
| Solo 穿透   | 10     | 8    | Bridge/Neck | 靠琴頸    |
| Jazz 圓潤   | 5      | 0~3  | Neck        | 靠琴頸    |
