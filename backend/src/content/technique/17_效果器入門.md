## 1. 訊號鏈順序

效果器的排列順序會影響最終的聲音。通用的順序：

```
🎸 吉他
 │
 ├─ 1. Tuner（調音器）         ← 最前面，確保乾淨的訊號
 ├─ 2. Compressor（壓縮器）    ← 穩定動態
 ├─ 3. Overdrive/Distortion    ← 增益/破音
 ├─ 4. EQ（等化器）            ← 調整頻率
 ├─ 5. Modulation（調變效果）  ← Chorus / Phaser / Flanger
 ├─ 6. Delay（延遲）           ← 回音
 ├─ 7. Reverb（殘響）          ← 空間感
 │
 🔊 音箱

口訣：「調 壓 破 等 調 延 殘」
```

---

## 2. 增益類效果

### Overdrive vs Distortion vs Fuzz

| 效果           | 破音程度 | 音色特點                         | 代表踏板                        |
| -------------- | -------- | -------------------------------- | ------------------------------- |
| **Overdrive**  | 低～中   | 溫暖、動態保留多、像音箱自然破音 | Ibanez Tube Screamer, Boss SD-1 |
| **Distortion** | 中～高   | 飽滿、壓縮感強、搖滾主力         | Boss DS-1, ProCo RAT            |
| **Fuzz**       | 極高     | 毛茸茸、復古、非常誇張的破音     | Electro-Harmonix Big Muff       |

```
聲音光譜：

  Clean ── Overdrive ── Distortion ── Fuzz

  乾淨     微破/溫暖     重破/飽滿     極破/毛躁

  Blues     Rock          Metal         Psychedelic
  Jazz                    Punk          Stoner Rock
```

## 3. 時間系效果

### Delay（延遲 / 回音）

```
你彈一個音，Delay 會在一段時間後重復播放那個音。

原音：     嗡
100ms 後：    嗡（稍小聲）
200ms 後：       嗡（更小聲）
300ms 後：          嗡（消失）

常用設定：
  Time：根據歌曲 BPM 設定（如 1/8 音符同步）
  Repeats/Feedback：2～4 次
  Mix：20～30%（不要蓋過原音）
```

### Reverb（殘響）

```
模擬空間的回聲效果。

  Room：小房間（短殘響，緊密）
  Hall：大廳（長殘響，空靈）
  Plate：金屬板振動模擬（明亮）
  Spring：彈簧（復古 Surf Rock 聲）

常用設定：
  Clean 段落：Room 或 Hall，Mix 30～40%
  Solo 段落：Hall，Mix 20～30%
  搖滾 Riff：少量或不加
```

### Chorus（合唱 / 和聲效果）

```
把你的吉他聲「複製」一份，讓兩份稍微走調 → 聽起來像兩把吉他同時彈。
音色變得更「寬」、更「厚」。

經典用途：Clean 音色的「水波紋」效果
代表聲音：Nirvana《Come As You Are》的 Clean 段
```

## 4. 工具類效果

### Compressor（壓縮器）

```
讓大聲的變小聲、小聲的變大聲 → 動態範圍縮小 → 聲音更「平均」。

用途：
  - 讓 Clean 音色更成熟穩定
  - 讓 Solo 的每個音都聽得清楚
  - 增加延音（Sustain）
```

### EQ（等化器）

```
調整特定頻率的音量。

  低頻（Bass）：增加 → 聲音更厚重；減少 → 聲音更乾淨
  中頻（Mid）：增加 → 穿透力強；減少 → 「scooped」空洞感
  高頻（Treble）：增加 → 更亮；減少 → 更暗

日系搖滾通常：Mid 不砍太多（保持穿透力）
```

### Noise Gate（噪音門）

```
自動在你不彈的時候關閉聲音，消除高 Gain 的「嘶嘶」雜音。
使用 Distortion 時幾乎必備。
```

---

## 5. 推薦的第一塊踏板

```
如果只買一塊：

  🥇 Overdrive（如 Boss SD-1 或 Ibanez TS Mini）
     → 最萬用，搭配任何音箱都能得到好聲音
     → 輕踩 = Crunch，重踩 = 接近 Distortion
     → 價格便宜（$30～60 USD）

如果買兩塊：
  🥇 Overdrive + 🥈 Delay
     → Delay 讓獨奏和 Clean 段落瞬間升級

如果買三塊：
  🥇 Overdrive + 🥈 Delay + 🥉 Tuner 踏板
     → 調音踏板是演出必備（靜音調音）
```
