## 1. 即興不是「亂彈」

```
亂彈：手指隨機移動，碰到什麼音就是什麼音
       → 大部分時候不好聽

即興：在**規則的框架內**即時創作旋律
       → 框架 = 音階 + 和弦 + 節奏
       → 在框架內自由探索 = 即興
```

### 即興的三要素

```
1. 音階（用什麼音）
   → 如 Am Pentatonic：A C D E G
   → 只用這 5 個音，怎麼排都好聽

2. 節奏（音的長短和時機）
   → 不是每個音都要一樣長
   → 有快有慢、有密有疏

3. 技巧（怎麼彈這些音）
   → 同一個音用 Picking vs Hammer-on vs Bend vs Slide
   → 效果完全不同
```

## 2. Step 1：用五聲音階即興

### 最簡單的即興方法

```
Step 1: 在 YouTube 搜尋 "A minor backing track" 或 "Am jam track"
Step 2: 播放 Backing Track
Step 3: 用 Am Pentatonic 把位 1（5 格位置）隨意彈

就這麼簡單。開始就好。
```

### 初學者的即興規則

```
Rule 1: 不要一直跑音階上下行
  ❌ A-C-D-E-G-E-D-C-A-C-D-E-G-E-D-C... (像在練習)
  ✅ 跳著來：A...E-D...G-A...C-D-E...稍停...G-E-D

Rule 2: 留空白
  ❌ 每一拍都塞滿音符
  ✅ 彈幾個音 → 停頓 → 再彈幾個音
  → 停頓是音樂的「呼吸」，跟說話一樣

Rule 3: 重複好的東西
  聽到自己彈的某段覺得「不錯」→ 重複它、稍微變化它
  這就是旋律的「主題」
```

## 3. Call and Response（呼應法）

```
像對話一樣：一句「問」，一句「答」。

Call（問）：     e|--5--8--5--------|
                              ↑ 停住，像在問問題

Response（答）： e|--8--5--8b10-----|
                              ↑ 推弦收尾，像在回答

練習：
  1. 即興彈一個短句（2～4 拍）
  2. 停頓 1～2 拍
  3. 彈另一個短句「回應」它
  4. 重複
```

## 4. Lick（常用樂句）

Lick 是 **預先學好的短樂句**，像是扑克牌手的「牌」。
即興時從你的 Lick 庫裡抽出適合的句子使用。

### 5 個入門 Lick（Am Pentatonic）

```
Lick 1（經典搖滾）：
e|--8b10--8--5--|
B|------------8--|

Lick 2（藍調轉音）：
e|--5--8--5------|
B|-----------8-5-|

Lick 3（推弦回放）：
e|--8b10r8--5----|

Lick 4（搥弦連音）：
e|--5h8p5--8-----|

Lick 5（滑音開頭）：
e|--/5--8--5-----|
B|-------------8-|
```

### 如何收集 Lick

```
1. 從喜歡的 Solo 中截取短句（2～4 拍）
2. 記住指法
3. 在不同的把位 / 不同的調性嘗試
4. 在 Backing Track 上「試裝」這個 Lick
5. 漸漸地，你的 Lick 庫會越來越大
```

## 5. Backing Track 練習法

### 選擇 Backing Track

```
YouTube 搜尋關鍵字：
  初學者：  "Am pentatonic backing track slow"
  進階：    "A minor blues backing track"
  搖滾風：  "rock backing track in A"

推薦頻道：
  Quist Backing Tracks
  Elevated Jam Tracks
  Justin Guitar Backing Tracks
```

### 練習流程

```
1. 播放 Backing Track（選一個 Am 的）
2. 前 2 分鐘：只用把位 1，慢慢探索
3. 中間 2 分鐘：嘗試插入你學過的 Lick
4. 最後 2 分鐘：嘗試 Call and Response
5. 錄下來 → 回放 → 找到好聽的段落 → 記住它
```

## 6. 發展個人風格

```
    技巧            品味           個性
     │               │              │
     ├ 音階           ├ 你聽什麼音樂  ├ 你偏好什麼情感
     ├ 推弦/揉弦      ├ 你喜歡哪些   ├ 你的節奏感
     ├ Legato/Picking │  吉他手      ├ 你的「空白」習慣
     ├ 速度           ├ 讓你起雞皮   │
     └ 節奏感         │  疙瘩的樂句  │
                     └──────────────┘
                           ↓
                      你的即興風格

風格不是「決定」出來的，是「長出來」的。
持續練習 + 持續聽音樂 + 持續錄音 → 風格自然形成。
```
