# 02 — 音程（Interval）

> 音程是音樂的「度量衡」。不懂音程，就像不懂公分和公尺，無法描述任何距離。

## 1. 什麼是音程？

**音程 = 兩個音之間的距離。**

```
C 到 D 的距離 = 2 個半音 = 大二度
C 到 E 的距離 = 4 個半音 = 大三度
C 到 G 的距離 = 7 個半音 = 純五度
```

在吉他上就是**格數**：

```
同一根弦上：
  0 → 2 格 = 2 個半音 = 大二度
  0 → 4 格 = 4 個半音 = 大三度
  0 → 7 格 = 7 個半音 = 純五度
```

## 3. 在指板上「看」音程

### 同弦上的音程

```
同一根弦上，每多 1 格 = 多 1 個半音

  0格  1   2   3   4   5   6   7
  ├──┼──┼──┼──┼──┼──┼──┤
  P1 m2 M2 m3 M3 P4 TT P5

例：6弦 5 格(A) → 6弦 7 格(B) = 2 格 = 大二度
```

### 跨弦的音程形狀

跨弦時，音程有固定的「圖形」（前提：不涉及 2-3 弦之間，那裡差 4 個半音而非 5 個）：

```
純五度(P5) = Power Chord 的形狀！

  6弦  5弦
  ●─────●     ← 差 2 格（同弦差 7 格 = 跨弦差 2 格）
  5格   7格
  A     E     ← A 到 E = 純五度

純八度(P8)：

  6弦  5弦  4弦
  ●─────────●  ← 跨 2 弦差 2 格
  5格        7格
  A          A  ← 同音，高八度


大三度(M3)：

  在普通弦組（如 6-5、5-4、4-3）：
  ●──●              ← 差 1 格往下（仔細想：5個半音-4個半音=差-1）
  不對，讓我用正確的方式表示：

  6弦 5格(A) 的大三度 = C# = 5弦 4格
  → 跨 1 弦，退 1 格

  小三度(m3)：
  6弦 5格(A) 的小三度 = C = 5弦 3格
  → 跨 1 弦，退 2 格
```

### 音程形狀速查圖

```
以 6 弦某格為根音（●），各音程在指板上的相對位置：

弦\格   -2   -1    0   +1   +2   +3   +4   +5   +7
6弦                 ●               M3   P4   TT   P5   m7
5弦          m3   M3   P4
4弦                                P8

（這只是示意，實際的跨弦關係在後續使用中會越來越直覺）
```

## 4. 「聽」音程 —— 耳朵訓練

### 為什麼要訓練耳朵？

```
看到譜 → 用眼睛
聽到歌 → 用耳朵

會「看」音程是閱讀能力
會「聽」音程是聽力能力

兩者兼備 → 才能「聽到一首歌就知道怎麼彈」
```

### 訓練方法

```
Step 1: 在吉他上彈兩個音（如 A 和 E = 純五度）
Step 2: 唱出來（先唱低音，再唱高音）
Step 3: 記住這個「距離」的感覺
Step 4: 多試幾次不同的根音（如 C→G、D→A），確認「純五度的感覺」是一致的

每天花 3 分鐘，一次只練一種音程。

建議順序：
  第 1 週：純五度（P5）
  第 2 週：純八度（P8）
  第 3 週：大三度（M3）
  第 4 週：小三度（m3）
```
