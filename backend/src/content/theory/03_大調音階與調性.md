## 1. 大調音階的公式

### 全全半全全全半

```
大調音階 = 7 個音，按照以下距離排列：

  全音 - 全音 - 半音 - 全音 - 全音 - 全音 - 半音
  (2)   (2)   (1)   (2)   (2)   (2)   (1)

以 C 大調為例：

  C ──2──→ D ──2──→ E ──1──→ F ──2──→ G ──2──→ A ──2──→ B ──1──→ C
  全       全       半       全       全       全       半

  C D E F G A B = C 大調的 7 個音
```

### 為什麼 C 大調最簡單？

```
C 大調的 7 個音 = C D E F G A B = 全部都是白鍵，沒有任何 #/♭。
所以音樂課本總是先教 C 大調。
```

## 2. 其他調的大調音階

公式是固定的（全全半全全全半），換個起始音就是不同的調：

### G 大調

```
G ──2──→ A ──2──→ B ──1──→ C ──2──→ D ──2──→ E ──2──→ F# ──1──→ G

G 大調 = G A B C D E F#  （有 1 個 #）
```

### D 大調

```
D ──2──→ E ──2──→ F# ──1──→ G ──2──→ A ──2──→ B ──2──→ C# ──1──→ D

D 大調 = D E F# G A B C#  （有 2 個 #）
```

### F 大調

```
F ──2──→ G ──2──→ A ──1──→ B♭ ──2──→ C ──2──→ D ──2──→ E ──1──→ F

F 大調 = F G A B♭ C D E  （有 1 個 ♭）
```

## 3. 在指板上的大調音階

### C 大調（開放把位）

```
e|--0--1-----3--|
B|--0--1-----3--|
G|--0-----2-----|
D|--0-----2--3--|
A|--0-----2--3--|
E|--0--1-----3--|
    C  D  E  F  G  A  B  C ...（音名依弦不同）
```

### C 大調（5 格把位 —— 更實用）

```
以 6 弦 8 格（C）為根音的大調音階指型：

e|----7--8--10----|
B|----8--10-------|
G|--7----9--10----|
D|--7----9--10----|
A|--7--8--10------|
E|--8--10---------|

這個形狀可以整個搬到不同格位 → 不同調的大調音階
搬到 5 格（A）→ A 大調
搬到 3 格（G）→ G 大調
```

## 4. 什麼是「調」（Key）?

### 調 = 一首歌用的音的集合

```
一首歌如果用了 C D E F G A B 這 7 個音 → 它在 C 大調
一首歌如果用了 G A B C D E F# 這 7 個音 → 它在 G 大調

「調」就像是「這首歌的字母表」。
在這個調裡的音 = 合法的音
不在這個調裡的音 = 通常不用（除非刻意製造效果）
```

### 主音（Tonic）

```
調的第一個音 = 主音 = 「回家」的感覺。

C 大調的主音 = C
G 大調的主音 = G

旋律和和弦最終會回到主音，這就是「解決感」。
當你聽一首歌覺得「結束了、安定了」→ 那個和弦通常就是主音。
```

---

## 5. 判斷一首歌的調

### 方法 1：最後一個和弦

```
大部分歌曲的最後一個和弦 = 主和弦 = 調名。

最後一個和弦是 G → 很可能是 G 大調（或 G 小調）
```

### 方法 2：最「安穩」的和弦

```
聽歌時，哪個和弦聽起來最「回家」、最「安定」？
那個就很可能是主和弦。
```

### 方法 3：觀察升降記號

```
沒有 # 或 ♭ → C 大調 / A 小調
1 個 # (F#) → G 大調 / E 小調
2 個 # (F# C#) → D 大調 / B 小調
1 個 ♭ (B♭) → F 大調 / D 小調
```

## 6. 音階的級數

每個音在音階中有自己的「級數」（degree）：

```
C 大調：C    D    E    F    G    A    B
級數：  1    2    3    4    5    6    7
別名：  主   上主  中   下屬  屬   下中  導

最重要的三個級數：
  1（主音）= 根基
  4（下屬） = 離開
  5（屬音）= 緊張，想回到 1
```
