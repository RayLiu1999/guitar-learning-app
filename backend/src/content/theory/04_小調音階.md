## 1. 自然小調的公式

### 全半全全半全全

```
自然小調 = 7 個音，按照以下距離排列：

  全音 - 半音 - 全音 - 全音 - 半音 - 全音 - 全音
  (2)   (1)   (2)   (2)   (1)   (2)   (2)

以 A 小調為例：

  A ──2──→ B ──1──→ C ──2──→ D ──2──→ E ──1──→ F ──2──→ G ──2──→ A

  A 小調 = A B C D E F G
```

等等 —— A B C D E F G？這不就是 C D E F G A B 的重新排列嗎？沒錯！

## 2. 關係大小調 ⭐

### A 小調 = C 大調的「影子」

```
C 大調：C  D  E  F  G  A  B
A 小調：A  B  C  D  E  F  G

完全相同的 7 個音！
只是「起點」不同 → 「感覺」就不同。

C 大調從 C 開始 → 明亮、快樂
A 小調從 A 開始 → 陰暗、悲傷
```

### 為什麼同樣的音會有不同的感覺？

```
關鍵在「主音」的位置。

C 大調以 C 為中心 → 其他音相對於 C 的距離是「大調」的距離
A 小調以 A 為中心 → 其他音相對於 A 的距離是「小調」的距離

特別是第 3 個音（三度音）：
  C 大調：C → E = 大三度（4 半音）→ 明亮
  A 小調：A → C = 小三度（3 半音）→ 陰暗

大三度 vs 小三度 = 大/小調情緒的根源。
```

### 關係大小調的公式

```
任何大調的第 6 個音 = 它的關係小調的主音

  C 大調的第 6 音 = A → 關係小調 = A 小調
  G 大調的第 6 音 = E → 關係小調 = E 小調
  D 大調的第 6 音 = B → 關係小調 = B 小調
  F 大調的第 6 音 = D → 關係小調 = D 小調

反過來：
  小調的第 3 個音 = 關係大調的主音

  A 小調的第 3 音 = C → 關係大調 = C 大調
```

## 3. 其他常見小調音階

### E 小調（電吉他常用！）

```
E ──2──→ F# ──1──→ G ──2──→ A ──2──→ B ──1──→ C ──2──→ D ──2──→ E

E 小調 = E F# G A B C D
關係大調 = G 大調（G A B C D E F#）← 一模一樣的音！
```

### F# 小調（GHOST 那首歌！）

```
F# ──2──→ G# ──1──→ A ──2──→ B ──2──→ C# ──1──→ D ──2──→ E ──2──→ F#

F# 小調 = F# G# A B C# D E
關係大調 = A 大調（A B C# D E F# G#）
```

## 4. 和聲小調

自然小調有一個「問題」：第 7 個音離主音差 2 個半音（全音），聽起來「解決感」不強。和聲小調把第 7 個音升高半音來解決這個問題。

```
A 自然小調：A  B  C  D  E  F  G   A
A 和聲小調：A  B  C  D  E  F  G#  A
                                ↑
                              升高半音！

G# → A 只差 1 個半音 → 解決感更強烈（半音有「吸引力」）
```

### 和聲小調的特色

```
第 6 和第 7 個音之間出現了 3 個半音的跳躍：
  F → G# = 增二度（3 個半音）

這個跳躍聽起來很「異國風」、「阿拉伯風」。

Yngwie Malmsteen、Ritchie Blackmore 等新古典金屬吉他手大量使用和聲小調。
```

## 5. 旋律小調（簡介）

```
上行：升高第 6 和第 7 個音（接近大調）
下行：回歸自然小調

A 旋律小調上行：A  B  C  D  E  F#  G#  A
A 旋律小調下行：A  G  F  E  D  C  B  A（= 自然小調）

爵士樂中常用（上行形式全程使用），搖滾中較少見。
暫時知道有這個東西就好。
```

## 6. 大調 vs 小調 —— 情緒對比

| 大調                      | 小調                      |
| ------------------------- | ------------------------- |
| 明亮、快樂                | 陰暗、悲傷                |
| 「陽光」                  | 「月光」                  |
| 唱歌像在「笑」            | 唱歌像在「嘆」            |
| 三度音 = 大三度（4 半音） | 三度音 = 小三度（3 半音） |
| Pop、Country、March       | Rock、Metal、Ballad       |

```
試試看：

在吉他上彈 C 大調音階（C D E F G A B C）→ 結尾停在 C → 感覺「完成了」

然後彈 A 小調音階（A B C D E F G A）→ 結尾停在 A → 感覺「有點憂鬱」

同樣的 7 個音，只是起點和終點不同，情緒就完全不同。
```
