## 1. 五聲音階的原理

### 七聲 → 五聲

```
A 自然小調（7 個音）：A  B  C  D  E  F  G
                       1  2  3  4  5  6  7

刪掉第 2 個(B)和第 6 個(F)：

A 小調五聲（5 個音）：A     C  D  E     G
                       1     3  4  5     7
```

### 為什麼刪掉 B 和 F？

```
B 和 F 是小調中的「半音鄰居」：
  B → C = 半音   ← 離太近，容易「打架」
  E → F = 半音   ← 離太近，容易「打架」

半音在旋律中會產生「摩擦感」/ 「緊張感」。
刪掉製造摩擦的音 → 剩下的音之間全是大二度或小三度 → 怎麼連怎麼跳都和諧。

這就是為什麼「五聲音階怎麼彈都好聽」。
```

## 2. 大調五聲與小調五聲

### 它們是同一組音！

```
C 大調五聲：C  D  E     G  A        （從大調刪掉 4 度和 7 度）
A 小調五聲：A     C  D  E     G     （從小調刪掉 2 度和 6 度）

同樣的 5 個音：A C D E G
起點不同 → 性格不同

C 大調五聲 → 明亮、歡快（鄉村/流行）
A 小調五聲 → 陰暗、搖滾（搖滾/藍調）
```

### 速查表

| 小調五聲  | = 關係大調五聲 | 共用的音    |
| --------- | -------------- | ----------- |
| Am Penta  | C Major Penta  | A C D E G   |
| Em Penta  | G Major Penta  | E G A B D   |
| Dm Penta  | F Major Penta  | D F G A C   |
| F#m Penta | A Major Penta  | F# A B C# E |

## 3. 藍調音（Blue Note）

在小調五聲音階中加入一個「額外的音」—— 減五度（♭5）：

```
A 小調五聲：   A  C  D     E  G
A 藍調音階：   A  C  D  D#  E  G
                        ↑
                     Blue Note!（D# = E♭）

這個 D# 聽起來「髒」、「彎」、「痛苦」—— 藍調的靈魂。
通常不會停在這個音上，而是作為經過音或推弦的目標。
```

### 在指板上

```
Am 五聲把位 1 + Blue Note：

e|--5--------8----|
B|--5--------8----|
G|--5--7----------|
D|--5--7--8-------|  ← 8 格 = Blue Note (D#/E♭)
A|--5--7----------|
E|--5--------8----|

只多了一個音（4弦8格），但味道完全不同：
  爬過 Blue Note → 像吉他在「嘆氣」
  推弦到 Blue Note → 像吉他在「哭」
```

## 4. 五個 Box 形狀與連接

技巧系列第 09 篇介紹了 5 個把位，這裡從「樂理」角度解釋它們之間的關係：

### 為什麼有 5 個把位？

```
五聲音階有 5 個音 → 每個音都可以作為一個 Box 的「起始音」。

Am Pentatonic：A(根) C(♭3) D(4) E(5) G(♭7)

Box 1 → 從 A 開始（5 格）
Box 2 → 從 C 開始（8 格）
Box 3 → 從 D 開始（10 格）
Box 4 → 從 E 開始（12 格）
Box 5 → 從 G 開始（3 格 / 15 格）
```

### 5 個 Box 覆蓋整個指板

```
Box 5    Box 1    Box 2    Box 3    Box 4    Box 5    Box 1
|  2~5  |  5~8  |  7~10  | 10~13 | 12~15 |  ...重複...
├───────┼───────┼────────┼───────┼───────┤

它們首尾相連，像拼圖一樣覆蓋 0～24 格。
```

### 連接方法

```
Box 1 → Box 2 的連接：

Box 1 結束的最高音 = Box 2 開始的最低音（或重疊區域）

在 1 弦上：
  Box 1：5--8
  Box 2：    8--10

8 格是共用音 → 從 Box 1 的 8 格滑到 Box 2 的 8 格（同音，然後繼續向上）

練習：
  用 Box 1 彈一段旋律 → 滑音到 Box 2 → 繼續彈 → 滑到 Box 3...
  這就是「在整個指板上自由移動」的基礎。
```

---

## 5. 五聲音階的練習建議

```
階段 1：只用 Box 1（可能需要 2～4 週）
  → 上下行閉眼都能跑

階段 2：加入 Box 2（1～2 週）
  → 練習 Box 1 → Box 2 的連接

階段 3：加入 Box 5（1～2 週）
  → Box 5 → Box 1 → Box 2 的三連跑

階段 4：全部 5 個 Box（持續練習）
  → 在整個指板上自由移動

階段 5：加入 Blue Note
  → 在每個 Box 中找到 Blue Note 的位置
```
