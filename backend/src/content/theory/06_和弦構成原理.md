# 06 — 和弦構成原理

> 音階是「線」，和弦是「面」。把音階的音堆疊起來，就誕生了和弦。

## 1. 三和弦的構造

### 最基本的公式

```
和弦 = 從音階中，每隔一個音取一個，取 3 個：

C 大調音階：C  D  E  F  G  A  B
            ↑     ↑     ↑
            取    跳    取    跳    取

C 和弦 = C + E + G  （根音 + 三度 + 五度）
```

### 四種三和弦

| 類型         | 公式                 | 音程組成        | 聽感 | 範例           |
| ------------ | -------------------- | --------------- | ---- | -------------- |
| **大三和弦** | 根 + 大三度 + 純五度 | 4 半音 + 3 半音 | 明亮 | C (C-E-G)      |
| **小三和弦** | 根 + 小三度 + 純五度 | 3 半音 + 4 半音 | 陰暗 | Cm (C-E♭-G)    |
| **減三和弦** | 根 + 小三度 + 減五度 | 3 半音 + 3 半音 | 不安 | Cdim (C-E♭-G♭) |
| **增三和弦** | 根 + 大三度 + 增五度 | 4 半音 + 4 半音 | 懸浮 | Caug (C-E-G#)  |

### 圖解

```
大三和弦（Major）：
  根音 ─── 大三度(4) ─── 純五度(7)
  C         E             G
  明亮 ✦

小三和弦（Minor）：
  根音 ─── 小三度(3) ─── 純五度(7)
  C         E♭            G
  陰暗 ✦

差異只在「三度音」差 1 個半音！
  大三度 = 4 半音 = 明亮
  小三度 = 3 半音 = 陰暗
```

## 2. 為什麼 Power Chord 不分大小調？

```
Power Chord = 根音 + 純五度（只有 2 個音）

C5 = C + G

沒有三度音！
  → 不知道是大三度還是小三度
  → 所以不分大調還是小調
  → 只有「力量」，沒有「情緒偏向」

這就是為什麼 Power Chord 的代號是「5」（如 A5、C5）
  → 只有根音和五度音

＋ Distortion 時只用 Power Chord 的原因：
  完整和弦（3 個音以上）在高度破音下會互相干擾 → 泥巴聲
  Power Chord 只有 2 個音 → 破音後依然乾淨
```

## 3. 在指板上理解和弦

### C 大三和弦在指板上的「堆疊」

```
以 5 弦 3 格（C）為根音：

5弦 3格 = C（根音）
4弦 2格 = E（大三度）
3弦 0格 = G（純五度）

  或者更高把位：
5弦 3格 = C
4弦 5格 = G（五度音，轉位後）
3弦 5格 = C（高八度根音）
  → 這就是封閉和弦的形狀！
```

### 用這個理解開放和弦

```
E 和弦為什麼按那個形狀？

  6弦 0格 = E  ← 根音
  5弦 2格 = B  ← 五度
  4弦 2格 = E  ← 根音（高八度）
  3弦 1格 = G# ← 大三度 ✦ 決定了它是「大調」
  2弦 0格 = B  ← 五度
  1弦 0格 = E  ← 根音（再高八度）

  只有 3 個不同的音：E、G#、B
  但分布在 6 根弦上，各種八度混合 → 聲音「厚」

Em 和弦呢？
  3弦 0格 = G  ← 小三度 ✦（比 G# 低一個半音）

  大和弦 → 小和弦：只需要把三度音降 1 個半音！
```

## 4. 順階三和弦（Diatonic Triads）

在一個調裡，以每個音為根音堆三和弦：

```
C 大調的 7 個順階三和弦：

級數：  I     ii    iii   IV    V     vi    vii°
和弦：  C     Dm    Em    F     G     Am    Bdim
性質：  大    小    小    大    大    小    減

大寫 = 大三和弦     小寫 = 小三和弦     ° = 減三和弦

記住這個模式：大-小-小-大-大-小-減
           I  ii iii IV  V  vi vii°
```

### 為什麼有些是大有些是小？

```
以 C 大調為例：

I = C-E-G    (C→E = 大三度 = 大和弦)
ii = D-F-A   (D→F = 小三度 = 小和弦)
iii = E-G-B  (E→G = 小三度 = 小和弦)
IV = F-A-C   (F→A = 大三度 = 大和弦)
V = G-B-D    (G→B = 大三度 = 大和弦)
vi = A-C-E   (A→C = 小三度 = 小和弦)
vii° = B-D-F (B→D = 小三度, D→F = 小三度 = 減和弦)

不是我們選擇的 —— 是音階的數學結構自動決定的！
```
