# 07 — 七和弦與延伸和弦

> 三和弦是黑白照片，七和弦是彩色照片。多加一個音，色彩就出來了。

## 1. 七和弦 = 三和弦 + 第七度音

```
三和弦：根 + 三度 + 五度（3個音）
七和弦：根 + 三度 + 五度 + 七度（4個音）

多加的第 7 個音讓和弦聽起來更「豐富」、更「有顏色」。
```

### 三種主要七和弦

| 類型         | 公式          | 組成      | 聽感           | 代號  |
| ------------ | ------------- | --------- | -------------- | ----- |
| **大七和弦** | 大三 + 大七度 | C-E-G-B   | 夢幻、爵士     | Cmaj7 |
| **屬七和弦** | 大三 + 小七度 | C-E-G-B♭  | 有張力、想解決 | C7    |
| **小七和弦** | 小三 + 小七度 | C-E♭-G-B♭ | 柔和、憂鬱     | Cm7   |

```
比較：
  C    = C-E-G        ← 明亮
  Cmaj7 = C-E-G-B     ← 明亮 + 夢幻
  C7   = C-E-G-B♭     ← 明亮 + 想解決
  Cm7  = C-E♭-G-B♭    ← 陰暗 + 柔和
```

## 2. 指板上的七和弦

### E 型 Dom7（最常用）

```
E7 開放和弦：
e|--0--|
B|--0--|
G|--1--|  ← G#（大三度）
D|--0--|  ← D（小七度！）← 這個音讓它變成「七」
A|--2--|
E|--0--|

A7 = E7 型搬到 5 格（封閉和弦）
B7 = E7 型搬到 7 格
```

### Am7（開放）

```
Am7：
e|--0--|
B|--1--|
G|--0--|  ← G（小七度）← 七度音
D|--2--|
A|--0--|
E|--x--|
```

## 3. 七和弦在搖滾中的使用

```
藍調搖滾：
  全部用屬七和弦（Dom7）
  A7 → D7 → E7 → A7   ← 12 Bar Blues

流行 / 日系：
  Cmaj7 → Am7 → Dm7 → G7  ← 爵士風流行

硬搖滾：
  通常不用七和弦（用 Power Chord + Distortion）
  但在 Clean 段落偶爾加入 m7 或 Maj7 做對比
```

## 4. Hendrix Chord（7#9）⭐

```
E7#9（The Hendrix Chord）：

e|--0--|
B|--0--|
G|--1--|  ← G#（三度）
D|--0--|  ← D（七度）
A|--2--|
E|--0--|
  + 加上 G 音（小三度 / #9）

= 同時有大三度(G#)和小三度(G)
= 不知道是大調還是小調 → 「矛盾」的感覺
= 聽起來又髒又酷

經典歌曲：Jimi Hendrix《Purple Haze》
```

## 5. 掛留和弦（Suspended Chords）

用二度或四度「取代」三度音：

```
Csus2 = C-D-G     （用二度 D 取代三度 E）
Csus4 = C-F-G     （用四度 F 取代三度 E）

因為沒有三度音 → 不分大小調
聽起來「懸浮」、「未解決」→ 通常會接回普通大/小和弦

C → Csus4 → C    ← 經典過門
Am → Asus2 → Am  ← 歌曲中常見的裝飾
```

### 在指板上

```
Asus2：               Asus4：
e|--0--|               e|--0--|
B|--0--|               B|--3--|  ← 四度(D)
G|--2--|               G|--2--|
D|--2--|               D|--2--|
A|--0--|               A|--0--|
E|--x--|               E|--x--|
```

## 6. 順階七和弦

```
C 大調的順階七和弦：

級數：  Imaj7   ii7    iii7   IVmaj7  V7     vi7    vii∅7
和弦：  Cmaj7   Dm7    Em7    Fmaj7   G7     Am7    Bm7♭5
性質：  大七    小七   小七   大七    屬七   小七   半減七

模式：大七-小七-小七-大七-屬七-小七-半減七
```
