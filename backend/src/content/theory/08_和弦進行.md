## 1. 級數系統（Roman Numeral Analysis）

### 用級數而不是和弦名

```
C 大調：C  Dm  Em  F   G   Am  Bdim
級數：  I  ii  iii IV  V   vi  vii°

為什麼用級數？因為同一個「進行」可以套到任何調：

  I - V - vi - IV

  C 大調：C  - G  - Am - F
  G 大調：G  - D  - Em - C
  D 大調：D  - A  - Bm - G

級數不變，音名隨調變。
用級數思考 = 一勞永逸。
```

## 2. 最常見的和弦進行 ⭐

### ① I - V - vi - IV（流行之王）

```
C 大調版：C - G - Am - F

使用這個進行的歌（不完全列表）：
  • Let It Be (Beatles)
  • No Woman No Cry (Bob Marley)
  • Someone Like You (Adele)
  • 小幸運 (田馥甄)
  • 無數動漫 OP / ED

這可能是人類歷史上被使用最多次的和弦進行。
```

### ② I - IV - V（搖滾三和弦引擎）

```
C 大調版：C - F - G
A 大調版：A - D - E

搖滾、藍調、鄉村的基礎。
只有 3 個和弦就能寫歌。
用 Power Chord 版本 → A5 - D5 - E5 → 純搖滾。
```

### ③ vi - IV - I - V（日系動漫/流行）

```
C 大調版：Am - F - C - G

也叫「小調開頭的流行進行」。
比 I-V-vi-IV 更「憂鬱」一些（因為小調和弦開頭）。

常見於：
  • 日本動漫 OP / 流行歌
  • 韓國 K-Pop
  • 許多中文流行歌
```

### ④ I - vi - IV - V（50年代進行 / Doo-Wop）

```
C 大調版：C - Am - F - G

復古風格，50～60 年代的經典。
也叫「Stand By Me 進行」。
```

### ⑤ ii - V - I（爵士進行）

```
C 大調版：Dm7 - G7 - Cmaj7

爵士樂最基本的進行。
ii → V 製造張力，V → I 解決張力。
```

## 3. 12 Bar Blues ⭐

藍調音樂的骨架，也是搖滾的DNA：

```
  | I    | I    | I    | I    |
  | IV   | IV   | I    | I    |
  | V    | IV   | I    | V    |

A 大調版：
  | A    | A    | A    | A    |
  | D    | D    | A    | A    |
  | E    | D    | A    | E    |

用 Dom7 和弦更道地：
  | A7   | A7   | A7   | A7   |
  | D7   | D7   | A7   | A7   |
  | E7   | D7   | A7   | E7   |

這 12 小節不斷重複 = 整首藍調歌曲的基礎。
搖滾、R&B、甚至流行都從這裡演化出來。
```

## 4. 和弦的「功能」

```
每個級數的和弦在進行中扮演不同的角色：

  I（主和弦/Tonic）    = 「家」—— 安穩、結束、回來
  IV（下屬/Subdominant）= 「出門」—— 離開家，但不遠
  V（屬和弦/Dominant）  = 「想回家」—— 緊張、有回到 I 的衝動

  大部分和弦進行就是：
    離開家(I) → 走遠(IV) → 想回家(V) → 回到家(I)

  ii 和弦常常替代 IV（功能相近）
  vi 和弦常常替代 I（功能相近，但偏悲）
```

## 5. 訓練方法：聽出和弦進行

```
Step 1: 找一首你喜歡的歌
Step 2: 聽低音（Bass）—— 通常就是和弦的根音
Step 3: 判斷低音的音名 → 推測和弦名
Step 4: 把和弦名轉成級數 → 看是不是哪個常見進行

工具推薦：
  - Chordify (chordify.net) → 自動辨識歌曲的和弦
  - Hooktheory (hooktheory.com) → 用級數分析大量流行歌

練習：
  每天分析一首歌的和弦進行，寫下進行的級數。
  連續做一週後，你會開始「預測」下一個和弦是什麼。
```
