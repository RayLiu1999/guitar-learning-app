## 1. CAGED 的核心想法

```
你已經會 5 種開放和弦：C, A, G, E, D

如果把這 5 種和弦的「形狀」搬到指板的不同位置（用食指取代弦枕），
就能在任何把位彈出任何和弦。

開放和弦 → 搬上去 → 封閉和弦
C型 → A型 → G型 → E型 → D型 → C型（循環）
```

---

## 2. 以 C 大調 和弦為例，5 個位置

### C 型（開放位置，0 格）

```
e|--0--|
B|--1--|
G|--0--|
D|--2--|
A|--3--|  ← 根音 C
E|--x--|

你已經知道的 C 和弦。根音在 5 弦 3 格。
```

### A 型（3 格，封閉）

```
把 A 和弦的形狀搬到 3 格：

e|--3--|  ← 食指橫按 3 格
B|--5--|
G|--5--|
D|--5--|
A|--3--|  ← 根音 C（5弦3格）
E|--x--|

跟 A 開放和弦的「形狀」一樣，只是整個往上移了。
```

### G 型（5 格）

```
e|--8--|
B|--8--|
G|--5--|
D|--5--|
A|--x--|
E|--8--|  ← 根音 C（6弦8格）

G 型比較難按，實際中常簡化。
```

### E 型（8 格）

```
e|--8--|  ← 食指橫按 8 格
B|--8--|
G|--9--|
D|--10-|
A|--10-|
E|--8--|  ← 根音 C（6弦8格）

最常用的封閉和弦型！
你在技巧訓練 05 篇學過的「E 型封閉和弦」就是這個。
```

### D 型（10 格）

```
e|--13-|
B|--13-|
G|--12-|
D|--10-|  ← 根音 C（4弦10格）（= 高把位）
A|--x--|
E|--x--|
```

---

## 3. CAGED 順序在指板上的排列

```
以 C 和弦為例，5 種型在指板上的分布：

格數：  0    3    5    8    10   12(=0)
型：   [C]  [A]  [G]  [E]  [D]  [C]
        └────┘────┘────┘────┘────┘ 循環！

每種型覆蓋一段指板。
5 種型加起來 → 覆蓋整個 0～12 格。
```

### 換成其他和弦

```
如果要找 A 和弦的 5 種位置：

格數：  0    2    5    7    10   12(=0)
型：   [A]  [G]  [E]  [D]  [C]  [A]

CAGED 中的字母順序是固定的：C → A → G → E → D → C → ...
A 和弦從 A 型開始，後面是 G-E-D-C 循環。
```

---

## 4. CAGED 與音階把位的對應

```
每個 CAGED 型對應一個五聲音階把位：

  C 型 → Box 5（五聲音階把位 5）
  A 型 → Box 1（五聲音階把位 1）← 最常用！
  G 型 → Box 2
  E 型 → Box 3
  D 型 → Box 4

這不是巧合 —— 和弦型和音階型本來就是同一個音階的不同「切片」。

知道和弦在哪個 CAGED 型 → 就知道應該用哪個音階把位即興。
```

---

## 5. 實際練習

### 練習 A：找出 G 和弦的 5 種位置

```
按照 CAGED 順序（從 G 型開始）：

G 型：0 格（開放 G 和弦）
E 型：3 格（超常用！G 的封閉和弦）
D 型：5 格
C 型：7 格
A 型：10 格

嘗試每個位置都按出來，從低到高彈一遍。
```

### 練習 B：沿著指板上行

```
1. 按 C 和弦（C 型，0 格）
2. 不換和弦名，滑到下一個型（A 型，3 格）
3. 繼續 → G 型(5格) → E 型(8格) → D 型(10格) → C 型(12格)

每個位置停下來刷一次，確認聲音正確。
```
