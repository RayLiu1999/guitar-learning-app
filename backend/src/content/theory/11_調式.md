## 1. 調式是什麼？

```
C 大調的 7 個音：C D E F G A B

如果以 C 為起點 → C Ionian（= 大調）
如果以 D 為起點 → D Dorian
如果以 E 為起點 → E Phrygian
以 F 為起點 → F Lydian
以 G 為起點 → G Mixolydian
以 A 為起點 → A Aeolian（= 自然小調）
以 B 為起點 → B Locrian

相同的 7 個音，7 個不同的起點 → 7 種不同的音程結構 → 7 種不同的情緒。
```

## 2. 七個調式速查表

| #   | 調式名         | 公式（半音）   | 情緒         | 一句話描述             |
| --- | -------------- | -------------- | ------------ | ---------------------- |
| 1   | **Ionian**     | 全全半全全全半 | 明亮快樂     | 就是大調，流行歌的基礎 |
| 2   | **Dorian**     | 全半全全全半全 | 稍暗但有希望 | 像小調但多了一絲亮光   |
| 3   | **Phrygian**   | 半全全全半全全 | 異域、緊張   | 西班牙/阿拉伯風味      |
| 4   | **Lydian**     | 全全全半全全半 | 夢幻、飄浮   | 像大調但更「外太空」   |
| 5   | **Mixolydian** | 全全半全全半全 | 力量但偏藍   | 像大調但更「搖滾」     |
| 6   | **Aeolian**    | 全半全全半全全 | 悲傷陰暗     | 就是自然小調           |
| 7   | **Locrian**    | 半全全半全全全 | 極度不安     | 幾乎不用（太不穩定）   |

## 3. 搖滾/金屬常用的調式

### Dorian（第 2 調式）

```
A Dorian：A B C D E F# G A
           ↑ 跟 A 自然小調的差別只在 F#（不是 F）

這個 F# 讓小調多了一絲「明亮」。

使用者：Santana、Pink Floyd
風格：搖滾/藍調即興
```

### Phrygian（第 3 調式）

```
E Phrygian：E F G A B C D E
             ↑ 第 2 音只差半音（E→F）→ 極其緊張

聽起來像「西班牙」或「阿拉伯」。

使用者：Metallica、Megadeth（金屬常用！）
風格：金屬 Riff
```

### Mixolydian（第 5 調式）

```
A Mixolydian：A B C# D E F# G A
               ↑ 跟 A 大調的差別只在 G（不是 G#）

像大調但更「接地氣」，有藍調的味道。

使用者：AC/DC、Lynyrd Skynyrd
風格：硬搖滾
```

## 4. 怎麼在指板上「用」調式？

```
最簡單的方法：

1. 確定和弦進行
2. 確定根音
3. 選擇對應的調式
4. 用相同的五聲/七聲音階指型，從對應的根音開始

例：如果和弦停在 Dm 上很久
  → 用 D Dorian 即興
  → D Dorian = C 大調的音，但以 D 為中心
  → 強調 D 音作為「回家的感覺」

理論上這就是你已經會的東西，
只是「重力中心」從 C 變成了 D。
```
