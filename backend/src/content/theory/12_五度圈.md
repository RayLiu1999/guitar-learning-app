# 12 — 五度圈（Circle of Fifths）

> 五度圈是音樂世界的地圖。所有調性、和弦進行、轉調的邏輯都在這個圓上。

## 1. 五度圈的構造

```
從 C 開始，每次往上跳一個純五度（7 個半音）：

C → G → D → A → E → B → F#(=G♭) → D♭ → A♭ → E♭ → B♭ → F → C

轉了一圈回到 C！排成圓形：

               C
          F         G
      B♭               D
   E♭                      A
      A♭               E
          D♭        B
             G♭/F#

順時針：每跳一格 = +1 個 #
逆時針：每跳一格 = +1 個 ♭
```

---

## 2. 用五度圈判斷調號

```
順時針方向（增加 #）：
  C = 0 個 #/♭
  G = 1 個 # (F#)
  D = 2 個 # (F#, C#)
  A = 3 個 # (F#, C#, G#)
  E = 4 個 # (F#, C#, G#, D#)
  B = 5 個 #

逆時針方向（增加 ♭）：
  F = 1 個 ♭ (B♭)
  B♭ = 2 個 ♭ (B♭, E♭)
  E♭ = 3 個 ♭ (B♭, E♭, A♭)
  A♭ = 4 個 ♭
```

### 內圈：關係小調

```
每個大調在五度圈的對應位置，往內圈看 = 關係小調：

  C 大調 ↔ A 小調
  G 大調 ↔ E 小調
  D 大調 ↔ B 小調
  F 大調 ↔ D 小調
  ...
```

---

## 3. 用五度圈理解和弦進行

```
在五度圈上相鄰的調 = 關係最近的調（共用最多的音）

  F ── C ── G
  ↑    ↑    ↑
  IV   I    V

I-IV-V 進行 = 在五度圈上「原地 + 左右各跳一格」！

這解釋了為什麼 I-IV-V 聽起來最自然：
  它們是五度圈上最近的鄰居，共用了最多的音。
```

### V → I 的解決

```
V → I = 在五度圈上「順時針跳一格」
這是音樂中最強烈的「解決」：

G7 → C  （G 是 C 的五度 → 回到 C = 回家）

藍調、搖滾、古典都建立在「五度解決」上。
```

---

## 4. 轉調

```
轉調 = 在五度圈上「搬家」

  C 大調 → G 大調 = 在五度圈上右移一格（升一個 #）
    → 非常平滑的轉調，聽眾幾乎不會注意到

  C 大調 → D 大調 = 在五度圈上右移兩格（升兩個 #）
    → 稍微有感覺的「升 key」

  C 大調 → F# 大調 = 在五度圈上的「對面」（差 6 個 #）
    → 非常戲劇化的轉調

越近的鄰居 = 越自然的轉調
越遠的位置 = 越戲劇化的轉調
```
