## 1. 功能和聲

每個和弦在進行中有一個「角色」（功能）：

```
┌─────────────────────────────────────────┐
│  Tonic（主功能）    = 安穩               │
│  Subdominant（下屬）= 移動、推進         │
│  Dominant（屬功能）  = 緊張、想解決回主   │
└─────────────────────────────────────────┘

C 大調中的分配：

  功能        和弦級數         和弦名
  Tonic       I, iii, vi       C, Em, Am
  Subdominant ii, IV           Dm, F
  Dominant    V, vii°          G, Bdim
```

### 和弦進行 = 功能的流動

```
典型的流動方向：

  T → SD → D → T
  I → IV → V → I
  C → F  → G → C

  「回家 → 出門 → 想回家 → 回家」

更複雜的進行本質上也是這個模式的變形。
```

## 2. V → I 解決（最重要的和聲概念）

```
V7 → I 是音樂中最強的「解決」：

G7(G-B-D-F) → C(C-E-G)

為什麼？
  B(第 7 級) → C(第 1 級) = 半音上行 → 解決
  F(第 4 級) → E(第 3 級) = 半音下行 → 解決

兩個音同時做半音移動到穩定的位置 → 極強的「到家了！」的感覺
```

## 3. 離調和弦

有時歌曲中會出現**不屬於當前調**的和弦。這些和弦叫做「離調和弦」。

### 範例

```
C 大調的順階和弦是：C Dm Em F G Am Bdim

但歌曲中出現了 E（大三和弦）而不是 Em（小三和弦）？
  → E 不在 C 大調的順階和弦中
  → 這就是「離調」
  → 它是從哪裡「借」來的？
```

## 4. 副屬和弦（Secondary Dominant）

```
V → I 是最強的解決。

如果我們把任何一個順階和弦暫時當作「I」，
然後在它前面加上它的「V」→ 這就是副屬和弦。

例：在 C 大調中，Am 的前面加上 E7：

  原本：... → Am → ...
  加強：... → E7 → Am → ...

E7 是 Am 的 V7（A 小調的屬七和弦）
→ E7 強力解決到 Am
→ 聽起來像「暫時轉到了 A 小調」然後馬上回來

記法：V7/vi  = 「vi 的 V7」= E7→Am（在 C 大調中）
```

### 常見的副屬和弦

```
C 大調中：

  V7/ii  = A7 → Dm     （Dm 暫時被當作「I」）
  V7/iii = B7 → Em
  V7/IV  = C7 → F      （注意：C7 而不是 C）
  V7/V   = D7 → G      （最常見！）
  V7/vi  = E7 → Am

D7 → G → C = 「到 G 的五度圈解決 → 到 C 的五度圈解決」
              連續兩次五度解決 → 非常有力！
```
