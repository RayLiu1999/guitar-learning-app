## 1. 和弦音即興（Chord Tone Soloing）

### 五聲音階即興的下一步

```
初級即興：用一個音階跑整首歌（如 Am Pentatonic 由頭到尾）
  → 安全但容易「漫無目的」

進階即興：跟隨和弦進行，在每個和弦上強調該和弦的組成音
  → 旋律「黏著」和弦走，聽起來更「有意義」
```

### 怎麼做？

```
和弦進行：Am - F - C - G

Am 和弦 → 強調 A、C、E（Am 的組成音）
F 和弦 → 強調 F、A、C（F 的組成音）
C 和弦 → 強調 C、E、G（C 的組成音）
G 和弦 → 強調 G、B、D（G 的組成音）

「強調」的意思：
  - 在和弦變換的第一拍落在和弦音上
  - 長音用和弦音
  - 短音可以用音階中的其他音
```

## 2. 目標音與經過音

```
目標音（Target Note）= 你「要到達」的音 = 通常是和弦音
經過音（Passing Tone）= 「路過」的音 = 音階中的非和弦音

例：Am 和弦上，目標音是 A、C、E

  你可以這樣的路線到達 A：
  G → A    （G 是經過音，A 是目標音）
  B → A    （B 是經過音，從上面靠近）
  G → G# → A  （G# 是半音經過音 → 更有張力！）

經過音「帶你去」目標音，就像走廊帶你去房間。
```

---

## 3. 在不同和弦上選擇不同的音階

```
初級：全曲用 Am Pentatonic
進階：每個和弦用不同的音階（= 調式思維）

Am 和弦 → A Dorian 或 A Aeolian
F 和弦 → F Lydian
C 和弦 → C Ionian
G 和弦 → G Mixolydian

但這太複雜了。實際的建議路徑：

Step 1: 先用五聲音階 + 強調和弦音（本篇的方法）
Step 2: 練到可以跟隨和弦進行後 → 試加入非五聲的音（第 6、7 個音）
Step 3: 再考慮為每個和弦切換調式
```

## 4. Inside vs Outside

```
Inside playing = 在「安全區」裡彈
  → 使用當前調性/和弦的音 → 和諧、穩定

Outside playing = 故意跳出安全區
  → 使用不在調性裡的音 → 緊張、意外

搖滾/藍調：90% Inside + 10% Outside（推弦推到非調性音 → 再推回來）
爵士：可能 50% Inside + 50% Outside

初學者先 100% Inside，等「安全區」練到本能反應再嘗試 Outside。
```

### 最簡單的 Outside 技巧

```
半音趨近（Chromatic Approach）：

  在目標音的前面加一個半音：

  要到 A → 先彈 G# → 滑到 A
  要到 E → 先彈 D# → 滑到 E

  G# 和 D# 可能不在你的音階裡 → 「Outside」
  但因為馬上解決到「Inside」的音 → 聽起來像「走味了一下馬上回來」
```
