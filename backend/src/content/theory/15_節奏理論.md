# 15 — 節奏理論

> 音符決定「彈什麼」，節奏決定「什麼時候彈」。節奏的影響力其實比音高更大。

## 1. 拍號

```
拍號 = 上面的數字 / 下面的數字

4/4 = 每小節 4 拍，四分音符為一拍
3/4 = 每小節 3 拍（華爾滋）
6/8 = 每小節 6 拍，八分音符為一拍（二大拍各含三小拍）
```

### 常見拍號

| 拍號    | 感覺           | 常見風格              |
| ------- | -------------- | --------------------- |
| **4/4** | 最自然、最常見 | 搖滾/流行/金屬（95%） |
| **3/4** | 圓舞曲、搖擺   | 華爾滋/部分 Ballad    |
| **6/8** | 悠盪的三連音感 | 藍調慢歌/民謠         |
| **2/4** | 進行曲         | 軍樂/某些 Punk        |

## 2. 強拍與弱拍

```
4/4 拍的自然重音：

  拍子：  1    2    3    4
  強度：  強   弱   中強  弱

  第 1 拍最強（「下拍」/ Downbeat）
  第 3 拍次強
  第 2、4 拍是弱拍

搖滾/流行的特色：反拍加重
  在 2 和 4 拍加上小鼓（Snare）→ 「嗒」

  鼓：  Kick  Snare  Kick  Snare
  拍：   1     2      3     4

  這個「2、4 拍的 Snare」就是搖滾的心跳。
```

## 3. Swing vs Straight

### Straight（直）

```
八分音符均等分配：

  |  ●   ●  |  ●   ●  |
     50% 50%

每兩個八分音符完全等長。
搖滾、金屬、流行幾乎都是 Straight。
```

### Swing（搖擺）

```
第一個八分音符「長」，第二個「短」：

  |  ●    ●|  ●    ●|
    ~67% ~33%

像三連音去掉中間那個：
  |  ●  _  ●|
     長  短

藍調、爵士、Shuffle Rock 用 Swing。

Stevie Ray Vaughan、B.B. King 的節奏就是 Swing。
```

## 4. 奇數拍（Odd Time Signatures）

### 5/4 拍

```
每小節 5 拍。聽起來像「多了一拍」：

  |  1   2   3   4   5  |

可以拆成 3+2 或 2+3：
  |  1  2  3 | 1  2  |    ← 3+2
  |  1  2 | 1  2  3  |    ← 2+3

經典範例：Dave Brubeck《Take Five》
```

### 7/8 拍

```
每小節 7 個八分音符。常拆成 2+2+3 或 3+2+2：

  |  1 2  3 4  5 6 7  |    ← 2+2+3

在日系搖滾/動漫音樂中偶爾出現，製造「不安份」的感覺。
凛として時雨、RADWIMPS 等樂團會用奇數拍。
```

## 5. Polyrhythm（複節奏）

```
兩種不同節奏同時進行：

3 對 2（最常見的複節奏）：

  右手：|  ●    ●    ●  |    ← 三等分
  左手：|  ●       ●     |    ← 二等分

聽起來像兩個獨立的律動在互相交錯。
金屬樂團 Meshuggah 和 Tool 大量使用複節奏。
```
