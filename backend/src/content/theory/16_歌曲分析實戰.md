## 1. 鏡頭法（三層分析）

```
🔭 宏觀（遠鏡頭）：
  「整首歌」的層面
  - 什麼調？大調/小調？
  - 什麼拍號？BPM？
  - 曲式結構（Intro-Verse-Chorus-Bridge-Outro）

🔍 中觀（中鏡頭）：
  「段落」的層面
  - 每段的和弦進行是什麼？
  - 和弦進行用級數怎麼表示？
  - 段落之間的關係（重複？對比？發展？）

🔬 微觀（近鏡頭）：
  「小節/樂句」的層面
  - 吉他 Riff 用了什麼音？什麼節奏？
  - 旋律怎麼跟和弦互動？
  - 用了什麼特殊技巧/效果？
```

## 2. 分析範例 1：經典搖滾（I-IV-V 模式）

### 假設歌曲資訊

```
調：A 大調
BPM：120
拍號：4/4
```

### 宏觀

```
結構：Intro(4) → Verse(8) → Chorus(8) → Verse(8) → Chorus(8) → Solo(8) → Chorus(8) → Outro(4)
（括號中 = 小節數）

整首歌只用了 I-IV-V 三個和弦。
```

### 中觀

```
Verse 和弦進行：
| A    | A    | D    | D    |
| A    | A    | E    | E    |
  I      I     IV     IV
  I      I     V      V

Chorus 和弦進行：
| D    | A    | E    | A    |
  IV     I     V      I

→ 完全是 I-IV-V 的搖滾三和弦引擎。
```

### 微觀

```
吉他用 Power Chord（A5, D5, E5）+ Palm Mute 八分音符律動。
Chorus 時打開（不悶）→ 力度對比。

技巧訓練連結：
  [技巧 03 Power Chord] + [技巧 04 Palm Mute] + [技巧 10 節奏進階]
```

## 3. 分析範例 2：日系流行（vi-IV-I-V 模式）

### 假設歌曲資訊

```
調：C 大調（= A 小調的關係大調）
BPM：160
拍號：4/4
```

### 中觀

```
Verse：
| Am   | F    | C    | G    |
  vi     IV     I     V

Chorus：
| F    | G    | Am   | Am   |
  IV     V     vi     vi

→ 典型的日系動漫進行。
   Verse 用 vi-IV-I-V（小調開頭 → 帶有憂鬱感）
   Chorus 用 IV-V-vi（4536 變形 → 能量堆疊到 vi）
```

## 4. 分析範例 3：GHOST（我們學過的歌）

```
調：F# 大調 / D# 小調（半音下降調弦）
BPM：91（八分音符體感 182）

和弦進行核心：
| F#5  | B5   | C#5  | D#5  |
  I      IV     V     vi

→ I-IV-V-vi = 經典流行搖滾進行！
  用 Power Chord → 不分大小調，但進行本身是大調感。

Verse 全程 Palm Mute → Chorus 打開
= 壓抑→釋放的動態設計。
```

## 5. 自己分析一首歌的步驟

```
Step 1: 找出調性（最後一個和弦 / 最「安穩」的和弦）
Step 2: 用 Chordify 或其他工具找出和弦進行
Step 3: 把和弦名轉成級數
Step 4: 看是不是某個常見進行（I-V-vi-IV? vi-IV-I-V? 12 Bar Blues?）
Step 5: 觀察段落之間的和弦進行有什麼變化
Step 6: 如果有異常和弦（離調） → 分析它可能是從哪裡「借」來的

每週分析一首歌，一個月後你對和弦進行的敏感度會大幅提升。
```
